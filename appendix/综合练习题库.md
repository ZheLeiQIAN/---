### 题目3（简答题）

**简述文献信息资源的四大类型及其特点，并特别说明缩微文献资料的特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献信息资源按记录手段和载体分为四大类型：

1. **刻写型**：以刻划、手工书写为记录手段，载体为自然物质或纸张。优点是原始性、真实性强；缺点是易损、难以复制。
2. **印刷型**：以印刷技术记录，载体为纸张。优点是阅读方便、流传广泛；缺点是体积大、存储空间有限。
3. **缩微型**：以光学记录方式记录在感光材料上。特点是**体积小（省98%空间）、保存期长（500年）**，但需借助机器阅读。
4. **视听型**：以电磁记录方式记录在电磁材料上。优点是动态直观、多媒体；缺点是需专用设备。

**解析**：本题考查文献四大类型的完整辨识和缩微型特点。注意缩微型"省98%空间、保存500年"是高频填空考点。

</details>

### 题目4（简答题）

**简述系统理论四大原则在信息资源建设中的应用。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

1. **整体性原则**：系统是有机整体，整体功能大于部分之和。应用：是信息资源保障体系的方法论基础，必须建立保障体系、明确系统目标、与社会环境相适应。
2. **联系性原则**：系统要素相互联系、相互作用。应用：为信息资源体系结构研究提供理论依据，核心是**共建共享**——图书馆间建立广泛合作关系，科学规划、分工协作。
3. **有序性原则**：系统结构有序、层次分明。应用：为信息资源组织提供理论指导，要求进行揭示加工整序、有序化布局、建立检索系统。
4. **动态性原则**：系统随时间变化发展。应用：要求信息资源建设适应社会信息需求变化，关注趋势、用户需求和外部环境。

**解析**：本题是高频简答题。需分别阐述四大原则的内涵和应用，注意联系性原则中的"共建共享"是核心考点。记忆口诀："整体建体系，联系促共享，有序管组织，动态应变化。"

</details>

**解析**：本题考查"辨识现象对应的理论"——文献半衰期反映的是文献老化的特征。需准确说出半衰期定义，并指出其属于文献老化理论。易错点：半衰期反映的是文献老化而非文献增长。

### 题目3（简答题）

**简述阅读环境的变化及其对信息资源建设的影响。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、阅读环境的变化**

1. **阅读形态变化**：从纸媒到多媒介、富文本，超文本成为阅读对象（文字+图片+音视频），数字化阅读率提高，从"读书"到"读屏"。
2. **阅读观念变化**：沉浸式阅读与社群社交化阅读相互融合，名著阅读影视化（通俗本/精简本/普及本/图画本/口袋本），听书率上升。
3. **阅读方式变化**：从线性到非线性阅读，从深阅读到浅阅读，呈现碎片化、跳跃式、快餐式特征。

**二、对信息资源建设的影响**

1. 需适应多媒介、富文本的阅读形态，加强数字资源建设
2. 需应对电子书馆配痛点（多终端适配、多元需求满足）
3. 需平衡深阅读与浅阅读的资源配比
4. 需关注听书等新型阅读方式，拓展资源类型

**解析**：本题考查环境扫描中的阅读环境变化。需从形态、观念、方式三个维度阐述变化，并结合信息资源建设实际分析影响。

### 题目4（简答题）

**简述开放获取（OA）的类型及其区别。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**开放获取（Open Access, OA）**是指科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权。

OA主要分为两种类型：

1. **金色OA（Gold OA）**：又称开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取。两种形式：开放期刊（整刊开放）和开放论文（部分开放）。可能有论文处理费APC。代表平台：BMC、PLoS、DOAJ。

2. **绿色OA（Green OA）**：又称开放存储（Open Access Archives），将研究成果存储到机构或学科知识库。两种形式：机构知识库和预印本数据库。一般免费自存储，但可能有 embargo（时滞期）。代表平台：arXiv、ChinaXiv。

**核心区别**：金色OA是"在OA期刊上发表"，绿色OA是"在知识库中存储"。

**解析**：本题考查OA的类型辨识。需先定义OA（强调"免费使用+保留著作权"），再从实现方式、形式、费用、时滞等维度比较金色与绿色OA。易错点：OA保留著作权，并非放弃著作权。

</details>

### 题目5（论述题）

**论述信息资源建设为什么属于经济学问题，并阐述成本-收益理论的四维调控。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、信息资源建设属于经济学问题的原因**

1. **资源稀缺性与需求无限性的矛盾**：信息资源建设需要投入大量的人力、物力、财力资源，而图书馆经费始终有限，必须在有限资源约束下进行选择和配置。
2. **存在多种成本类型**：信息资源建设涉及会计成本（购书经费、设备费用、人员工资等实际发生的成本）和机会成本（选择某种方案而放弃的其他方案的收益，如选择购买A学科图书而放弃B学科图书的潜在收益）。
3. **追求效益最大化**：信息资源建设的根本目的是利用，需通过科学调控使有限资源产生最大效益，这正是经济学研究的核心命题。

**二、成本-收益理论的四维调控**

通过调控手段协调信息资源在以下四个维度的分布，使有限资源产生最大效益：
1. **时间维度**：把握信息资源的时效性，合理安排采购、更新与剔除的时序
2. **空间维度**：统筹不同地区、不同图书馆间的资源布局，避免重复建设
3. **部门维度**：协调馆内各部门及馆际间的分工协作
4. **数量维度**：合理确定各类型资源的收藏规模与复本配置

**解析**：本题是论述题考点。需从"为什么是经济学问题"和"如何调控"两个层面展开，结合会计成本与机会成本的区分，体现理论联系实际的分析能力。

</details>

### 题目2（简答题）

**简述读者决策采购（PDA）按读者参与途径分类的4种类型。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

读者决策采购（PDA）按读者参与途径可分为4种类型：

1. **通过馆际互借**：当读者提出馆际互借请求时，图书馆直接购买该文献而非借阅。如普渡大学设定150美元/3年内的阈值
2. **通过馆藏/联合目录**：将书商书目导入馆藏或联合目录，读者检索后触发荐购。如伊利诺伊大学设定300美元/1年内的阈值
3. **通过电子书点击浏览**：读者浏览电子书达到设定条件后触发自动购买。如丹佛大学设定5分钟/10页→短期租借→3次触发购买
4. **通过网上/线下书店**：读者在书店发现所需文献后，通过IP认证等方式由图书馆购买。如水牛城大学模式

**解析**：本题考查PDA的4种实施方式。关键在于区分"读者参与的途径"不同。记忆口诀："互借目录点击浏览，书店四种途径。"

</details>

### 题目3（简答题）

**简述电子资源定价模式的3大类11种，并举例说明。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

电子资源定价模式可归纳为3大类共11种：

**一、多重价格模式（4种）**——将数据库分割成不同产品，赋以不同价格，实现市场细分：
1. 全库价格+学科库价格（如爱思唯尔SD：全库+24学科库=25种价格）
2. 现期数据价格+回溯数据价格（如《自然》系列：现刊限最近5年）
3. 全库价格+单品种价格（如德古意特电子书）
4. 数据库价格+印本价格（如IOP模式）

**二、捆绑价格模式（3种）**——将两种或两种以上相关产品捆绑打包定价出售：
1. 印本捆绑（最典型常见，如爱思唯尔SD最低订购额）
2. 新产品捆绑（打包新产品，涨幅偏高）
3. 内容重复产品捆绑（如EBSCO ASP与BSP重复率20%）

**三、歧视性价格模式（4种）**——针对不同图书馆需求和支付能力制定不同收费价格：
1. 按学校/机构规模分级（如汤森路透SCI：6个级别）
2. 按现有订购量分级（如Wiley）
3. 按使用统计分级（如IEEE按下载量4级）
4. 按使用需求分级（如SciFinder按并发用户数定价）

**解析**：本题是高频简答与论述题。需准确记忆3大类名称、每类细分数量及典型实例。记忆口诀："多重四，捆绑三，歧视四，共计十一。"

</details>

8. **语种标准**：以馆藏文献的语种为标准
9. **保障标准**：以文献的保障要求为标准

**一、以系统理论四大原则奠定方法论基础**

1. **整体性原则**要求将信息资源建设视为有机整体，建立有明确目标、与社会环境相适应的信息资源保障体系。这要求图书馆跳出单馆视角，从国家或地区整体出发规划资源布局，使整体功能大于部分之和。

3. **有序性原则**要求对信息资源进行揭示、加工、整序，建立有序化布局和完善的检索系统，使用户能够便捷地获取所需资源。
4. **动态性原则**要求信息资源建设适应社会信息需求的变化，关注技术趋势、用户需求和外部环境，及时调整建设策略。

**二、以成本-收益理论四维调控实现效益最大化**

在系统理论框架下，需通过成本-收益理论的四维调控协调资源分布，使有限资源产生最大效益：

### 综合论述题2

**面对数字时代阅读环境的变化（第2章）与电子资源定价模式的复杂化（第3章），图书馆应如何优化其电子与数据资源采选策略？请结合第4章电子资源采购方式与第5章数据资源建设进行论述。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

数字时代，阅读环境的深刻变化与电子资源定价模式的复杂化给图书馆资源建设带来双重挑战。优化电子与数据资源采选策略，需从需求响应、采购模式选择、品质评价和数据资源拓展四个方面系统推进。

**一、响应阅读环境变化，调整资源结构**

第2章指出阅读环境发生三方面变化：形态上从纸媒到多媒介、富文本；观念上沉浸式与社群社交化融合、听书率上升；方式上从线性到非线性、从深阅读到浅阅读。据此，图书馆应：
1. 加强数字资源建设，适应"读书"到"读屏"的转变，增加电子书、多媒体资源配比
2. 应对电子书馆配痛点，解决多终端适配和多元需求满足问题
3. 平衡深阅读与浅阅读资源配比，既保障学术研究需要的系统文献，也满足碎片化阅读需求
4. 拓展听书等新型资源类型，回应用户需求变化

**二、应对定价模式复杂化，优化采购方式**

第3章归纳的电子资源定价模式有3大类11种（多重价格4种、捆绑价格3种、歧视性价格4种），给图书馆带来成本压力和选择难题。应对策略：
1. 充分理解各类定价模式的逻辑，利用多重价格模式实现市场细分下的灵活选择
2. 警惕捆绑价格模式（尤其是印本捆绑）增加采购成本、限制选择自由
3. 善用歧视性价格模式中的分级机制，争取符合本馆规模和使用统计的优惠价格
4. 依据资源品质评价三角度（资源内容、检索系统功能、提供商信誉服务）进行采选决策，确保"物有所值"

**三、合理选择电子资源采购方式**

第4章指出电子资源采购有国家采购、集团采购、单馆自主采购三种方式，三者互补：
1. 对综合性和基础性的公共基础信息，争取纳入国家采购，缓解数字鸿沟
2. 对综合性、多学科全文数据库，积极参与以DRAA为代表的集团采购，享受批量优惠、节省谈判成本
3. 对专业性、边缘性学科数据库，采用单馆自主采购，形成本馆特色资源体系

**四、拓展数据资源建设新领域**

第5章揭示数据资源建设是数字时代的前沿领域，应：
1. 建设特色数据库，遵循"特、精、全、省"四要求，打造人无我有的独特资源
2. 推进数据价值化三阶段（资源化→资产化→资本化），使数据从可用到可计量再到可交易
3. 遵循FAIR原则（可发现、可访问、可互操作、可重用）推进科学数据开放共享
4. 区分公共数据（政务数据、科学数据）与私密数据（企业数据、个人数据），分类建设管理

### 综合论述题3

**试论述馆藏文献复选与剔除（第6章）在馆藏体系建设中的意义，并结合第1章特色化建设理念与第2章文献老化理论说明其理论依据。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏文献复选与剔除是馆藏体系建设的下游关键环节，对保持馆藏活力、优化馆藏结构具有重要意义。其理论基础可追溯到第1章特色化建设理念和第2章文献老化理论。

**一、复选与剔除的内涵**

复选是在藏书补充过程后一阶段，对馆内入藏书刊的再次选择，与初选构成辩证统一过程（苏联格利科尔耶夫60年代提出，斯多利亚洛夫1979年系统阐述）。剔除是根据一定原则和标准，将长期滞留书架、利用率极低或已失去利用价值的文献从馆藏体系中分离出来并分别处理的结果性操作。

**二、理论依据**

1. **文献老化理论（第2章）**：文献内容会随时间老化，半衰期反映某学科现时尚在利用的文献中较新一半的发表时间。基础学科半衰期长、老化慢，应用学科半衰期短、老化快。这为复选提供了科学依据——通过半衰期测定法和书龄法，可对不同学科文献的取舍做出有理论支撑的判断，及时剔除已老化的文献。

2. **特色化建设理念（第1章）**：特色化建设要求"人无我有、人有我多、人多我精"，建设稀缺型和完备型特色资源。复选与剔除是实现特色化的必要手段——通过剔除与馆藏政策不符、内容陈旧的文献，集中资源建设本馆特色馆藏，使馆藏体系更具独特性和系统性。

3. **信息资源建设根本目的是利用（第1章）**：信息资源建设的根本目的是利用而非收藏，收藏只是前提。滞架时限标准（最佳标准）和使用频率标准正是以"是否被利用"为判断依据，将长期未被使用的文献剔除或调配，使有限空间和经费服务于真正被需要的资源。

**三、复选标准与方法的应用**

复选需综合运用9种标准（内容、外形、书龄、滞架时限、使用频率、复本量、替代、语种、保障）和9种方法（经验判断法、滞架时限法、书龄法、半衰期测定法、目录比较法、用户评议法、内容层次判断法、数学计算方法、外形判断方法）。其中滞架时限标准是确定流通书库文献去留的最佳标准，半衰期测定法体现了文献老化理论的直接应用。

**四、剔除处置与馆藏优化**

剔除后的文献按6种方式处置（调拨、捐赠、出售、置换、报损、报废），其中调拨、置换等方式可使文献在馆际间合理流动，结合第6章贮存图书馆的3种模式（专有贮存、地区合作贮存、国家贮存），可为罕用文献找到合适的归宿，既节省本馆空间，又保障了文献的整体可获得性。

**五、意义总结**

复选与剔除使馆藏体系始终保持动态优化状态：剔除老化、低利用率文献，腾出空间和经费；强化特色馆藏建设；促进馆际资源流动与共享；最终实现馆藏"有进有出、新陈代谢"的良性循环，使信息资源建设真正服务于利用这一根本目的。

**解析**：本题考查复选与剔除的跨章节理论联系。需将第6章的复选标准方法与第1章特色化理念、第2章文献老化理论贯通，说明复选既有理论依据又有实践价值，体现"理论指导实践、实践深化理论"的认识逻辑。

</details>

## 第1章 信息资源类型

> 考点覆盖：文献四大类型、出版物（ISBN 13位/ISSN 8位/CN号）、特种文献7类型、缩微文献特点（省98%空间/保存500年）、五大基本理念、数据资源建设理念（5条）

### 题目1（简答题）

**简述文献信息资源的四大类型及其特点，并特别说明缩微文献资料的特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献信息资源按记录手段和载体分为四大类型：

1. **刻写型**：以刻划、手工书写为记录手段，载体为自然物质或纸张。优点是原始性、真实性强；缺点是易损、难以复制。
2. **印刷型**：以印刷技术记录，载体为纸张。优点是阅读方便、流传广泛；缺点是体积大、存储空间有限。
3. **缩微型**：以光学记录方式记录在感光材料上。特点是**体积小（省98%空间）、保存期长（500年）**，但需借助机器阅读。
4. **视听型**：以电磁记录方式记录在电磁材料上。优点是动态直观、多媒体；缺点是需专用设备。

**解析**：本题考查文献四大类型的完整辨识和缩微型特点。注意缩微型“省98%空间、保存500年”是高频填空考点。

</details>

### 题目2（填空/选择题）

**图书的国际标准书号ISBN为____位，连续出版物的国际标准刊号ISSN为____位，国内统一刊号为____号。下列哪组文献均属于特种文献？（ ）**

A. 图书、期刊、报纸　B. 科技报告、会议文献、学位论文　C. 刻写型、印刷型、缩微型　D. 专著、丛书、年鉴

<details>
<summary>点击查看答案与解析</summary>

**答案：**13；8；CN；**B**

**解析**：

ISBN是国际标准书号，共**13位**（2007年起由10位升至13位），用于标识图书；ISSN是国际标准刊号，共**8位**数字编码，用于标识连续出版物；CN号是国内统一刊号，格式为CN XX-XXXX/YY，是中国期刊的身份标识。注意区分ISBN（13位，图书）与ISSN（8位，连续出版物）。

特种文献是指出版发行和获取途径都比较特殊的科技文献，包括科技报告、会议文献、学位论文、标准文献、专利文献、科技档案、政府出版物**七大类**（记忆口诀：“科会学标专档政”）。选项A中的图书、期刊、报纸属于常规出版物；选项C是按记录手段分类的文献类型；选项D中的专著、丛书属于图书，年鉴属于连续出版物。

### 题目1（简答题）

**简述系统论四大原则及其对信息资源建设的指导。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

系统论四大原则及对信息资源建设的指导：

1. **整体性原则**：系统是由相互联系的要素构成的有机整体，整体功能大于各要素功能之和。对信息资源建设的指导：要把信息资源建设作为一个整体系统来规划，使各类型资源（实体与数字）、各环节（采选、组织、服务、评价）协同配合，避免各自为政。

2. **联系性原则**：系统各要素之间、系统与外部环境之间存在相互联系、相互制约。对信息资源建设的指导：要处理好信息资源建设各要素间的关系，以及与出版、技术、用户需求、政策环境等的联系，推进馆际协作与资源共享。

3. **有序性原则**：系统内部结构是有序的、有层次的。对信息资源建设的指导：要建立层次化的馆藏体系（核心藏书、辅助藏书、特色藏书），按学科、专题有序组织，形成有序的馆藏结构。

4. **动态性原则**：系统是发展变化的，处于动态平衡。对信息资源建设的指导：馆藏要不断新陈代谢，剔除老化文献、补充新文献，适应用户需求和外部环境变化动态调整馆藏结构。

**解析**：本题考查系统论四大原则及其应用。需准确说出每个原则的内涵，并对应说明对信息资源建设的指导意义。记忆口诀：“整体建体系，联系促共享，有序管组织，动态应变化。”

### 题目3（名词解释）

**请解释“半衰期”的概念，并说明它反映了文献的什么特征。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**半衰期（Half-life）**是指某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的。

半衰期反映的是**文献老化**的特征，属于文献老化理论的核心概念。不同学科的半衰期不同：基础学科（如地理学16.1年、数学10.5年）半衰期长，文献老化慢；应用学科（如生物医学3.0年、冶金学3.9年）半衰期短，文献老化快。

半衰期对信息资源建设的意义：为评价和选择文献信息资源提供理论依据，为优化馆藏文献资源结构提供依据，为制定科学合理的文献工作原则提供依据。

**解析**：本题考查“半衰期反映文献的什么特征”——文献半衰期反映的是文献老化的特征。需准确说出半衰期定义，并指出其属于文献老化理论。注意与布拉德福定律（文献分布）、普赖斯曲线（文献增长）等理论区分。易错点：半衰期反映的是文献老化而非文献增长。

### 题目5（论述题）

**论述信息资源建设为什么属于经济学问题，并阐述成本-收益理论的四维调控。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、信息资源建设属于经济学问题的原因**

1. **资源稀缺性与需求无限性的矛盾**：信息资源建设需要投入大量的人力、物力、财力资源，而图书馆经费始终有限，必须在有限资源约束下进行选择和配置。
2. **存在多种成本类型**：信息资源建设涉及会计成本（购书经费、设备费用、人员工资等实际发生的成本）和机会成本（选择某种方案而放弃的其他方案的收益，如选择购买A学科图书而放弃B学科图书的潜在收益）。
3. **追求效益最大化**：信息资源建设的根本目的是利用，需通过科学调控使有限资源产生最大效益，这正是经济学研究的核心命题。

**二、成本-收益理论的四维调控**

通过调控手段协调信息资源在以下四个维度的分布，使有限资源产生最大效益：
1. **时间维度**：把握信息资源的时效性，合理安排采购、更新与剔除的时序
2. **空间维度**：统筹不同地区、不同图书馆间的资源布局，避免重复建设
3. **部门维度**：协调馆内各部门及馆际间的分工协作
4. **数量维度**：合理确定各类型资源的收藏规模与复本配置

**解析**：本题是论述题考点。需从“为什么是经济学问题”和“如何调控”两个层面展开，结合会计成本与机会成本的区分，体现理论联系实际的分析能力。

</details>

### 题目2（简答题）

**简述开放获取（OA）的类型及其区别。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**开放获取（Open Access, OA）**是指科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权。

OA主要分为两种类型：

1. **金色OA（Gold OA）**：又称开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取。两种形式：开放期刊（整刊开放）和开放论文（部分开放）。可能有论文处理费APC。代表平台：BMC、PLoS、DOAJ。

2. **绿色OA（Green OA）**：又称开放存储（Open Access Archives），将研究成果存储到机构或学科知识库。两种形式：机构知识库和预印本数据库。一般免费自存储，但可能有 embargo（时滞期）。代表平台：arXiv、ChinaXiv。

**核心区别**：金色OA是“在OA期刊上发表”，绿色OA是“在知识库中存储”。

**解析**：本题考查OA的类型辨识。需先定义OA（强调“免费使用+保留著作权”），再从实现方式、形式、费用、时滞等维度比较金色与绿色OA。易错点：OA保留著作权，并非放弃著作权。

</details>

### 题目3（简答题）

**简述CC知识共享协议的四要素、六种组合及与OA的关系。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**CC（Creative Commons）知识共享协议**由非营利组织Creative Commons于2001年推出，提供灵活的著作权授权方式。

**一、四要素**
1. 署名（BY, Attribution）：使用时须标明原作者
2. 非商业性使用（NC, NonCommercial）：不得用于商业目的
3. 禁止演绎（ND, NoDerivatives）：不得修改、转换原作品
4. 相同方式共享（SA, ShareAlike）：衍生作品须采用相同协议

**二、六种组合**（均包含BY署名）
1. CC BY：仅署名（最宽松）
2. CC BY-SA：署名+相同方式共享
3. CC BY-ND：署名+禁止演绎
4. CC BY-NC：署名+非商业性使用
5. CC BY-NC-SA：署名+非商业+相同方式共享
6. CC BY-NC-ND：署名+非商业+禁止演绎（最严格）

此外还有**CC0**：作者放弃全部版权，使作品进入公有领域。

**三、CC协议与OA的关系**
OA强调“免费使用+保留著作权”，而保留著作权后的授权方式通常通过CC协议实现。其中**CC BY是OA出版最常用的许可证**，CC协议是实现OA开放授权的工具。

**解析**：本题考查CC协议的完整内容。需准确说出四要素、六种组合（注意都含BY），并阐述CC与OA的关系。记忆口诀：“四要素BY/NC/ND/SA，六种组合都含BY，CC0放弃入公有，OA最常用是CC BY。”

</details>

### 题目4（简答题）

**简述图书馆信息资源建设面临的主要著作权问题（5类）。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

图书馆信息资源建设面临的著作权问题主要有5类：

1. **翻印/影印外国出版物**：未经授权翻印或影印外国出版物，涉及复制权侵权问题
2. **采购非法出版物**：采购盗版或侵权出版物，本身构成侵权行为
3. **文献复制**：超出合理使用范围的文献复制服务涉及著作权侵权

### 题目5（名词解释）

**请解释“FAIR原则”的内涵，并说明数据资源按公共/私密分类的类型。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、FAIR原则**

**FAIR原则**是科学数据开放共享应遵循的基本原则，于**2014年**提出，要求科学数据实现四个目标，四个目标层层递进：

1. **F - Findable（可发现）**：数据可被搜索和定位，用户能够找到所需数据
2. **A - Accessible（可访问）**：数据可被获取和访问，用户能够访问数据内容
3. **I - Interoperable（可互操作）**：数据可与其他系统互操作，不同系统和平台间能够交换和使用数据
4. **R - Reusable（可重用）**：数据可被重复使用，数据具有足够的描述和许可信息支持再利用

**二、数据资源类型（按公共/私密分类）**

1. **公共数据**：
   - **政务数据**：政府公务活动过程中生产的数据（如医疗健康、农业、区域大数据资源）
   - **科学数据**：基础研究、应用研究、试验开发等产生的数据（含原始数据和衍生数据）

2. **私密数据**：
   - **企业数据**：国有控股企业经营活动中形成的数据（如电信、银行、央企数据）
   - **个人数据**：与已识别或可识别自然人相关的数据（直接标识/间接标识）

**解析**：FAIR原则需准确记忆四个字母的含义及2014年提出，四者层层递进：先能发现，再能访问，进而能互操作，最终实现可重用。数据资源类型注意：政务数据和科学数据属于公共数据，企业数据和个人数据属于私密数据。

### 综合论述题3

**试论述馆藏文献复选与剔除（第6章）在馆藏体系建设中的意义，并结合第2章特色化建设理念与第2章文献老化理论说明其理论依据。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏文献复选与剔除是馆藏体系建设的下游关键环节，对保持馆藏活力、优化馆藏结构具有重要意义。其理论基础可追溯到第2章特色化建设理念和第2章文献老化理论。

**一、复选与剔除的内涵**

复选是在藏书补充过程后一阶段，对馆内入藏书刊的再次选择，与初选构成辩证统一过程（苏联格利科尔耶夫60年代提出，斯多利亚洛夫1979年系统阐述）。剔除是根据一定原则和标准，将长期滞留书架、利用率极低或已失去利用价值的文献从馆藏体系中分离出来并分别处理的结果性操作。

**二、理论依据**

1. **文献老化理论（第2章）**：文献内容会随时间老化，半衰期反映某学科现时尚在利用的文献中较新一半的发表时间。基础学科半衰期长、老化慢，应用学科半衰期短、老化快。这为复选提供了科学依据——通过半衰期测定法和书龄法，可对不同学科文献的取舍做出有理论支撑的判断，及时剔除已老化的文献。

2. **特色化建设理念（第2章）**：特色化建设要求“人无我有、人有我多、人多我精”，建设稀缺型和完备型特色资源。复选与剔除是实现特色化的必要手段——通过剔除与馆藏政策不符、内容陈旧的文献，集中资源建设本馆特色馆藏，使馆藏体系更具独特性和系统性。

3. **信息资源建设根本目的是利用（第1章）**：信息资源建设的根本目的是利用而非收藏，收藏只是前提。滞架时限标准（最佳标准）和使用频率标准正是以“是否被利用”为判断依据，将长期未被使用的文献剔除或调配，使有限空间和经费服务于真正被需要的资源。

**三、复选标准与方法的应用**

复选需综合运用9种标准（内容、外形、书龄、滞架时限、使用频率、复本量、替代、语种、保障）和9种方法（经验判断法、滞架时限法、书龄法、半衰期测定法、目录比较法、用户评议法、内容层次判断法、数学计算方法、外形判断方法）。其中滞架时限标准是确定流通书库文献去留的最佳标准，半衰期测定法体现了文献老化理论的直接应用。此外还需运用6种评价方法（专家评价、用户评议、统计分析、书目核对、引文分析、成本收益分析）对馆藏进行整体评价。

**四、剔除处置与馆藏优化**

剔除后的文献按6种方式处置（无偿调拨、捐赠、出售、置换、报损、报废），其中调拨、置换等方式可使文献在馆际间合理流动，结合第6章贮存图书馆的3种模式（专有贮存、地区合作贮存、国家贮存），可为罕用文献找到合适的归宿，既节省本馆空间，又保障了文献的整体可获得性。

**五、意义总结**

复选与剔除使馆藏体系始终保持动态优化状态：剔除老化、低利用率文献，腾出空间和经费；强化特色馆藏建设；促进馆际资源流动与共享；最终实现馆藏“有进有出、新陈代谢”的良性循环，使信息资源建设真正服务于利用这一根本目的。

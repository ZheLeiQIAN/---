# 附录：重点概念速查表

> 本速查表严格遵循新考点清单（共40考点），按章节排列，每章分“必考概念”与“重要概念”两个表格，便于考前快速检索与记忆。

## 第1章 信息资源类型

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 文献信息资源四大类型 | 按记录手段和载体分为刻写型、印刷型、缩微型、视听型 | 四大类常考多选题 |
| 出版物 | 以一定方式出版发行的文献资料，按出版形式分为图书、连续出版物、特种文献 | 按出版形式的三大分类 |
| 特种文献 | 出版发行和获取途径都比较特殊的文献资料，一般为科技文献 | 别称“灰色文献”，七大类型 |
| 缩微型文献 | 以光学记录方式将文献缩小记录在感光材料上的文献形式 | 体积小省98%空间、保存500年 |
| 信息资源建设五大基本理念 | 国家信息化战略组成部分/图书馆基础性工作/根本目的是利用/“拥有”与“存取”并重/共建共享既是观念也是过程 | 根本目的是利用（非收藏） |
| 数据资源建设理念 | 数据资源建设应遵循的5条基本理念 | 从战略、基础、目的、方式、共享五维度理解 |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| ISBN | 国际标准书号，13位数字，用于标识图书 | 2007年起由10位升至13位 |
| ISSN | 国际标准刊号，8位数字编码，用于标识连续出版物 | 与ISBN区分 |
| CN号 | 国内统一刊号，格式CN XX-XXXX/YY | 中国期刊的身份标识 |
| 连续出版物 | 具有统一题名、定期或不定期连续出版、计划无限期出版的出版物 | 期刊、报纸、年报等 |
| 科技报告 | 描述科学技术研究成果或进展的文献 | 内容详实、数据完整、报道速度快 |
| 会议文献 | 在学术会议上宣读或发表的论文 | 学术性强、内容新颖、反映前沿 |
| 学位论文 | 为取得学位而撰写的学术论文 | 内容系统、论述详细、具有独创性 |
| 标准文献 | 由权威机构批准发布的规范性文献 | 规范性强、具有法律约束力 |
| 专利文献 | 专利申请与审批过程中产生的文献 | 技术性、法律性、公开性 |
| 科技档案 | 科研生产活动过程中形成的记录 | 原始性、真实性、保密性 |
| 政府出版物 | 政府机构出版的文献 | 权威性、政策性、综合性 |

## 第2章 相关理论

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 系统论四大原则 | 整体性、联系性、有序性、动态性，是系统论的核心原则 | 指导信息资源建设的整体规划 |
| 整体性原则 | 系统是由相互联系的要素构成的有机整体，整体功能大于各要素功能之和 | 信息资源建设要整体统筹 |
| 联系性原则 | 系统各要素之间、系统与外部环境之间存在相互联系、相互制约 | 处理要素间与环境的关系 |
| 有序性原则 | 系统内部结构是有序的、有层次的 | 建立层次化的馆藏体系 |
| 动态性原则 | 系统是发展变化的，处于动态平衡 | 馆藏动态调整、新陈代谢 |
| 特色化建设 | 图书馆根据自身定位和资源优势，形成具有独特性的馆藏体系 | 理念：人无我有/人有我多/人多我精；层次：稀缺型/完备型；类型6种 |
| 信息生命周期 | 信息从生成到失效的全过程：生成→采集→存储/组织/加工→利用→维护→失效/剔除 | 信息资源建设就是完成一个生命周期的过程 |
| 布拉德福定律 | 将期刊按载文量分为三区，每区文献数约为总数的1/3，期刊数成等比数列（公比≈5） | P1:P2:P3 = 1:n:n²（n≈5） |
| 半衰期（Half-life） | 某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的 | 反映文献老化的特征 |
| 文献老化 | 文献内容的老化和文献载体的老化 | 内容老化是研究重点 |
| 价值论选书观 | 杜威、海恩斯主张选择科学或艺术价值高的出版物，以文献内容价值为选择依据 | 片面性：不考虑读者需求 |
| 需要论选书观 | 普尔·克特强调藏书补充的实用性，以用户需要为依据 | 片面性：忽视社会阅读导向 |
| 使用概率论 | 阮冈纳赞提出入藏每本书是否被利用是随机事件，概率越大表示被利用可能性越大 | 引入数学方法，客观但操作难 |
| 成本-收益理论 | 通过调控手段协调信息资源在时间、空间、部门、数量上的分布，使有限资源产生最大效益 | 经济学命题，四维调控 |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 稀缺型特色馆藏 | 收藏独有的、稀见的特色资源，形成独特性 | 特色化建设层次之一 |
| 完备型特色馆藏 | 在某一领域形成系统、完整的收藏 | 特色化建设层次之一 |
| 普赖斯曲线 | 文献增长与时间成指数函数关系：F(t) = ae^(bt) | 指数增长模型 |
| 综合计分法 | 麦高文从文献知识价值和社会需求两方面综合评价 | 辩证但难实行 |
| 社会调查选择论 | 维拉德依据对读者的社会、心理研究结果进行选书 | 按年龄、教育、职业分组 |
| 会计成本 | 实际发生的成本（购书经费、设备费用、人员工资） | 成本类型之一 |
| 机会成本 | 选择某种方案而放弃的其他方案的收益 | 成本类型之一 |

## 第3章 环境与相关概念

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 阅读环境变化 | 阅读形态、阅读观念、阅读方式三方面的变化 | 形态（纸媒→多媒介）/观念（沉浸式与社交化融合）/方式（线性→非线性、深→浅） |
| 开放获取（OA） | 科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权 | 保留著作权+免费使用 |
| 金色OA | 开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取 | 开放期刊+开放论文；可能有APC |
| 绿色OA | 开放存储（Open Access Archives），将研究成果存储到机构或学科知识库 | 自存储；可能有 embargo 时滞 |
| CC（知识共享协议） | 由非营利组织Creative Commons于2001年推出的版权授权协议体系，提供灵活的著作权授权方式 | 四要素+六种组合+CC0 |
| CC0 | 作者放弃全部版权，使作品进入公有领域 | CC协议的特殊形式，不属于六种组合 |
| 信息资源建设5类著作权问题 | 翻印影印、采购非法出版物、文献复制、馆藏数字化、数据库建设五类 | 5类常考简答题 |
| 读者决策采购（PDA/DDA） | 将读者的阅读需求量化成一定指标，作为图书馆文献采购和馆藏建设的决策依据 | 也称DDA（需求驱动采购）、POD（按需购买） |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| CC四要素 | 署名(BY)、非商业性使用(NC)、禁止演绎(ND)、相同方式共享(SA) | 六种组合均包含BY；CC BY是OA最常用许可证 |
| 第三空间（Third Place） | 由Ray Oldenburg提出，指居住（第一空间）、工作（第二空间）之外的购物休闲空间 | 图书馆定位为第三空间 |
| 期刊危机（Serials Crisis） | 连续出版物支出增长远超通胀和图书馆预算，导致图书馆无力负担 | 支出占比从25%增至近40% |
| 智慧图书馆 | 基于微服务架构的统一云服务开放平台，以数据共享为纽带连接各功能模块 | “智能”重技术，“智慧”重人本 |
| 图书馆发展变化5方面 | 预算压力与期刊危机/第三空间/用户地位变化/资源需求变化/技术革新 | PDA是用户地位变化的体现 |

## 第4章 文献资源采选

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 文献采选3依据 | 文献资源采选的三个基本依据：馆藏政策、读者需求、经费条件 | 与电子资源4依据对比，电子多出“资源品质”“定价模式” |
| 五级等级结构 | 完整级→研究级→大学级→基础级→最低级，由高到低的藏书建设等级体系 | 五级递降，完整级最高、最低级在范围之外 |
| 完整级 | 完整齐全搜集某专题领域所有知识记录为目标的藏书方案 | 最高级别，国家图书馆核心领域适用 |
| 研究级 | 以满足独立研究需要为目标的藏书方案 | 特藏应达到研究级 |
| 大学级 | 以满足大学生和个人自学大学课程需要为目标的藏书方案 | 高校图书馆基础配置 |
| 基础级 | 以介绍人们认识不同专业领域为目标的藏书方案 | 普及性，县级馆适用 |
| 最低级 | 在藏书范围之外的藏书方案 | 非馆藏重点，最低层次 |
| 国家图书馆采选特点 | 采选方针“中文求全、外文求精”，采选四级别全面/着重/适当/不宜 | 中文求全+外文求精 |
| 纲目订书 | 图书馆将藏书补充计划按入藏文献范围、内容、出版社、价格等列成纲目提交给书商，书商按“纲目”直接寄书 | 退书率≤10%，签一次合同连续供书 |
| 长期订购 | 保证多卷和连续出版物完整入藏的手段，一次手续长期有效 | 可享10%-30%优惠 |
| 集中式交换 | 将可供交换文献送交交换中心，由中心集中交换 | 最基本最常用，非营利，仅收少许手续费 |
| 采选四阶段 | 选书阶段→订购阶段→验收阶段→补缺阶段 | 顺序固定：选书/订购/验收/补缺 |
| 馆配商选择5种方式 | 公开招标/邀请招标/竞争性谈判/询价采购/单一来源采购+采购备案 | 中文→公开招标；外文→竞争性谈判；数据库→采购备案 |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 馆藏政策 | 图书馆根据其性质、任务和读者需求制定的文献资源建设指导性文件 | 采选的根本依据，文献与电子采选共有 |
| 读者需求 | 读者对文献信息的实际需要 | 文献采选的直接依据；电子采选称“用户需求” |
| 经费条件 | 图书馆可用于文献采选的经费预算及财务状况 | 文献采选特有依据 |
| 全面采选 | 中文各学科各类型文献全面入藏 | 国家图书馆级别一 |
| 着重采选 | 古籍善本、名人手稿、珍稀文献 | 国家图书馆级别二 |
| 适当采选 | 外文文献适时调整范围和重点 | 国家图书馆级别三 |
| 不宜采选 | 范围之外文献 | 国家图书馆级别四 |
| 提类订书 | 按类别集中订购 | 按学科类别批量采购 |
| 全数订购 | 某出版社文献全部订购 | 保证完整性 |
| 集中订购 | 集中向某一供应商订购 | 统一采购降低成本 |
| 租借订购 | 先租后买，试用满意再正式采购 | 降低采购风险 |
| 双边式交换 | 双方直接互寄文献 | 涉及范围广、周期长 |
| 多边式交换 | 各馆送书目到交换书目中心，中心分发供求书目 | 交换对象不确定 |
| 捐赠注意事项 | 捐赠协议/质量把关/产权归属/登记入藏/特殊要求/感谢回执 | 6项注意事项 |

## 第5章 电子与数据资源采选

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 电子资源采选4依据 | 电子资源采选的四个基本依据：馆藏政策、用户需求、资源品质、定价模式 | 比文献采选3依据多出“资源品质”“定价模式”两项 |
| 四级数字资源结构 | 永久保存级→服务级→镜像级→链接级，按数字资源重要程度和保存方式分为四个级别 | 唯一性递减，稳定性递减 |
| 永久保存级馆藏 | 保存价值已鉴定、用途得到认可、具有唯一性，且仅定位在某特定图书馆，其他地方不可获得 | 最高级别，唯一性是核心 |
| 服务级馆藏 | 对用户有用且必须用的虚拟馆藏；本馆用户十分需要，但贮存在其他数字图书馆服务器上 | 虚拟馆藏，存于他处 |
| 镜像级馆藏 | 其他数字图书馆馆藏的拷贝，地址可能移动，经常不可存取 | 拷贝性质，不稳定 |
| 链接级馆藏 | 在任何地方均可寄居的信息资料 | 最低级别 |
| 多重价格 | 同一资源以不同组合方式定价，包括全库+学科库、现期+回溯、全库+单品种、数据库+印本4种 | 定价模式第1大类，4种 |
| 捆绑价格 | 将多种产品捆绑销售定价，包括印本捆绑、新产品捆绑、内容重复产品捆绑3种 | 定价模式第2大类，印本捆绑最典型 |
| 歧视性价格 | 根据买方不同情况差别定价，包括按规模/订购量/使用统计/使用需求分级4种 | 定价模式第3大类，4种 |
| 国家采购 | 对与国家经济及社会发展战略目标相关的、教学科研人员和社会公众普遍需要的基础性、综合性数字资源，利用公共财政（政府出资）购买的采购模式 | 3型：统购/补贴/平台 |
| 集团采购 | 若干图书馆自愿组成集团，共同推举谈判代表与电子资源提供商谈判价格与使用条款 | DRAA是典型代表 |
| DRAA | 高校图书馆数字资源采购联盟（Digital Resource Acquisition Alliance of Chinese Academic Libraries） | 中国高校集团采购核心组织 |
| 单馆自主采购 | 各馆根据本馆需要，自发与信息资源提供商谈判价格并自行签署购买使用协议 | 适用于价格不高而利用率较高的数据库 |
| 数据资源化 | 数据收集与清洗、数据标注与描述、数据组织与集成，使数据成为可用资源 | 数据价值化第一阶段（使数据可用） |
| 数据资产化 | 数据确权、数据估值、数据管理，使数据成为可计量的资产 | 数据价值化第二阶段（使数据可计量） |
| 数据资本化 | 数据保值增值、数据流通交易、数据开发利用，使数据成为可交易的资本 | 数据价值化第三阶段（使数据可交易） |
| FAIR原则 | 科学数据开放共享应实现可发现（Findable）、可访问（Accessible）、可互操作（Interoperable）、可重用（Reusable），4个目标层层递进 | 2014年提出 |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 资源品质评价三角度 | 从资源内容（全/广/新）、检索系统功能、提供商信誉与服务三个角度评价电子资源品质 | 电子采选4依据之一 |
| 统购型 | 国家利用中央经费统一购买电子资源，全国每台联网计算机均可免费访问 | 中央经费全额，如冰岛 |
| 补贴型 | 政府支付一部分，另一部分由各图书馆支付 | 政府+各馆分担，如加拿大 |
| 平台型 | 国家投资建立统一采购与访问平台，平台费由国家支付，资源购买费由基层馆负担 | 国家出平台费+基层出资源费，如英国 |
| 特色数据库 | 图书馆根据本馆特色馆藏、特定用户需求或地域文化特色，自建的具有独特性的数据库 | 网络资源建设的核心内容；建设四要求：特/精/全/省 |
| 政务数据 | 主要存在于政府电子政务系统，是政府公务活动过程中生产的数据所形成的数据资源 | 公共数据 |
| 科学数据 | 自然科学、工程技术科学等领域通过基础研究、应用研究、试验开发等产生的数据，以及通过观测监测、考察调查、检验检测等方式取得的原始数据及其衍生数据 | 公共数据 |
| 企业数据 | 国有控股企业经营活动中形成的数据资源 | 私密数据，如电信/银行/央企 |
| 个人数据 | 与已识别或可识别自然人相关的数据 | 私密数据，直接标识/间接标识 |

## 第6章 组织管理

### 必考概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 复选（Re-selection） | 在藏书补充过程后一阶段，对馆内入藏书刊的再次选择；与初选构成辩证统一过程 | 苏联格利科尔耶夫60年代提出，斯多利亚洛夫1979系统阐述 |
| 剔除（Weeding/Deselection） | 根据一定原则和标准，将长期滞留在书架上、利用率极低或已失去利用价值的文献从馆藏体系中分离出来，并按不同情况分别处理 | 复选的结果性操作 |
| 滞架时限（Shelf-stand Time） | 文献在两次流通之间（含最后一次流通至今）滞留在书架上未被使用的时间长度 | 确定流通书库中文献馆内调配和去留的最佳标准 |
| 半衰期（Half-life） | 某学科被利用的文献总量中，一半文献失去利用效率所经历的时间 | 1960年伯顿和凯普勒提出；复选方法之一 |
| 分类排架法 | 按文献本身内容所属的学科体系来排列藏书的方法；由分类号和辅助号组成分类排架号 | 主要的馆藏文献排列方法 |
| NDPP | 国家数字科技文献资源长期保存体系（National Digital Preservation Program），我国科技文献长期保存机构 | 保障科技文献长期可持续访问 |
| 贮存图书馆（Depository Library） | 为解决收藏空间紧张、节省管理成本、提高利用率而设立的专门收藏利用率相对较低文献资源的图书馆 | 3种模式：专有/地区合作/国家贮存 |

### 重要概念

| 概念 | 一句话定义 | 关键要点 |
|------|----------|----------|
| 初选（Initial Selection） | 补充过程开始阶段对馆外出版物的入藏选择 | 与复选构成辩证统一过程 |
| 馆藏文献布局 | 将图书馆入藏的全部文献，按一定标准划分为若干部分，建立各种功能的书库、阅览空间 | 建筑空间角度，实质是空间位置上的划分 |
| 塔式垂直布局 | 按图书馆建筑结构，以垂直方向层层叠置的布局方式 | 高层书库，节省占地 |
| 水平展开布局 | 按图书馆建筑结构，以水平方向展开的布局方式 | 低层大面积，便于阅览 |
| 主题排架法 | 按文献主题排列藏书的方法 | 按内容排架的一种 |
| 固定排架法 | 按出版物的固定编号排架；编号由库室号、书架号、层格号、书位号组成 | 按形式排架，位置固定 |
| 登记号排架法 | 按文献登记顺序排列的排架方法 | 按形式排架，一书一号 |
| 色标排架法 | 以颜色标识区分文献的排架方法 | 按形式排架，辅助性视觉标识 |
| 无偿调拨 | 主管部门组织，有计划地将馆藏文献无偿调拨给其他图书馆 | 处置方式之一，强调无偿性 |
| 置换 | 以本馆多余文献与其他图书馆交换所需文献 | 处置方式之一 |
| 专有贮存图书馆 | 某一图书馆系统或机构独立设立的贮存图书馆，收藏本系统利用率较低的文献 | 独立性强，服务本系统 |
| 地区合作贮存图书馆 | 若干图书馆联合设立的贮存图书馆，服务于特定地区的成员馆 | 合作共享，服务地区 |
| 国家贮存图书馆 | 国家级机构设立的贮存图书馆，收藏全国范围内呈缴本或低利用率文献 | 国家层面，服务全国 |

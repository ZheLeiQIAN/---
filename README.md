## 章节结构说明

每个章节均包含 **8个标准模块**：

1. **模块定位** — 学科位置、知识功能、考试权重、考点覆盖
2. **知识图谱** — 考点结构化呈现
3. **核心概念** — 必考概念 + 重要概念表格
4. **重点公式/算法** — 关键结构、数字、分类体系
5. **方法比较** — 对比表格，辨析易混淆知识点
6. **考试聚焦** — 必考考点、重要考点、题型预测
7. **模拟练习** — 名词解释/填空/选择/简答/论述题（含答案）
8. **自测清单** — 概念掌握检查、方法应用检查、易错点提醒

## 使用指南

### 备考复习路径

1. **通读章节**：按第1→6章顺序通读，建立知识框架
2. **概念速查**：使用 `appendix/重点概念速查表.md` 快速回忆概念
3. **练习巩固**：使用 `appendix/综合练习题库.md` 检验掌握程度
4. **自测查漏**：利用各章末尾"自测清单"逐项自检
5. **易错点强化**：重点关注各章"易错点提醒"表格

### 考前冲刺

1. 复习各章"重点公式/算法"部分的记忆口诀
2. 复习"关键数字速记"表格
3. 重点练习"模拟练习"中的简答题和论述题

### 审查验证

- 使用 `appendix/审查流程文档.md` 了解审查方法
- 使用 `appendix/考点对照核对清单.md` 逐条核对考点覆盖情况

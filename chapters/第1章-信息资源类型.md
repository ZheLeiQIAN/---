## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **文献信息资源四大类型** | 按记录手段和载体分为刻写型、印刷型、缩微型、视听型 | 四大类是常考多选题 |
| **专利权三属性** | 独占性、时间性、地域性 | 常考多选题 |
| **专利三要求** | 新颖性、创造性、实用性 | 常考多选题 |
| **缩微型文献** | 以光学记录方式将文献缩小记录在感光材料上的文献形式 | 体积小省98%空间、保存500年 |
| **信息资源建设根本目的** | 信息资源的利用（收藏和保存只是前提） | 理念题高频考点 |
| **系统理论四大原则** | 整体性、联系性、有序性、动态性 | 常考简答题 |
| **特色化建设** | 图书馆信息资源具有独特风格，与其他图书馆不同的特点 | 理念：人无我有、人有我多、人多我精 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **ISBN** | 国际标准书号，13位数字，用于标识图书 | 与ISSN区分 |
| **ISSN** | 国际标准刊号，8位数字编码，用于标识连续出版物 | 与ISBN区分 |
| **CN号** | 国内统一刊号，格式CN XX-XXXX/YY | 中国期刊的身份标识 |
| **专利类型与期限** | 发明专利20年、实用新型10年、外观设计10年 | 期限数字易考 |
| **信息资源保障体系** | 基于系统整体性原则，建立的有明确目标、与社会环境相适应的信息资源系统 | 整体性原则的应用 |
| **共建共享** | 图书馆之间建立广泛合作关系，科学规划、分工协作、共同建设、相互提供利用 | 联系性原则的核心应用 |
| **稀缺型特色资源** | 图书馆独具特色的部分馆藏 | 特色化建设层次之一 |
| **完备型特色资源** | 对某一学科或某一类型文献具有系统性收藏的馆藏 | 特色化建设层次之一 |

## 【重点公式/算法】

### 1. 专利保护期限

| 专利类型 | 保护期限 |
|----------|----------|
| 发明专利 | 20年 |
| 实用新型 | 10年 |
| 外观设计 | 10年 |

### 2. 特色化建设目标公式

```
特色化建设理念 = 人无我有 + 人有我多 + 人多我精

数量目标：尽可能完整系统地收集、入藏的完备程度
质量目标：有一定深度、反映前沿、结构合理
类型目标：虚拟资源、文献资源、重点藏书与核心期刊
```

### 3. 系统理论四大原则应用

```
整体性原则 → 必须建立信息资源保障体系、明确目标、适应环境
联系性原则 → 共建共享、统筹安排、馆际互借
有序性原则 → 揭示加工整序、有序化布局、建立检索系统
动态性原则 → 关注趋势、用户需求、外部环境
```

**记忆口诀**：整体建体系，联系促共享，有序管组织，动态应变化。

## 【模拟练习】

### 题目1（简答题）

**简述文献信息资源的四大类型及其特点，并特别说明缩微文献资料的特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献信息资源按记录手段和载体分为四大类型：

1. **刻写型**：以刻划、手工书写为记录手段，载体为自然物质或纸张。优点是原始性、真实性强；缺点是易损、难以复制。
2. **印刷型**：以印刷技术记录，载体为纸张。优点是阅读方便、流传广泛；缺点是体积大、存储空间有限。
3. **缩微型**：以光学记录方式记录在感光材料上。特点是**体积小（省98%空间）、保存期长（500年）**，但需借助机器阅读。
4. **视听型**：以电磁记录方式记录在电磁材料上。优点是动态直观、多媒体；缺点是需专用设备。

**解析**：本题考查文献四大类型的完整辨识和缩微型特点。注意缩微型"省98%空间、保存500年"是高频填空考点。

</details>

**专利类型分为____、____、____三种，其中发明专利保护期限为____年，实用新型和外观设计为____年。专利权的三属性是____、____、____；专利三要求是____、____、____。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

专利类型分为**发明**、**实用新型**、**外观设计**三种。发明专利保护期限为**20**年，实用新型和外观设计为**10**年。

专利权的三属性是**独占性**、**时间性**、**地域性**。

专利三要求是**新颖性**、**创造性**、**实用性**。

**解析**：本题是高频填空/选择题。专利期限（20/10/10年）、三属性（独占/时间/地域）、三要求（新颖/创造/实用）均为必考数字和列表，需准确记忆。

</details>

### 题目3（简答题）

**简述系统理论四大原则在信息资源建设中的应用。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

1. **整体性原则**：系统是有机整体，整体功能大于部分之和。应用：是信息资源保障体系的方法论基础，必须建立保障体系、明确系统目标、与社会环境相适应。
2. **联系性原则**：系统要素相互联系、相互作用。应用：为信息资源体系结构研究提供理论依据，核心是**共建共享**——图书馆间建立广泛合作关系，科学规划、分工协作。
3. **有序性原则**：系统结构有序、层次分明。应用：为信息资源组织提供理论指导，要求进行揭示加工整序、有序化布局、建立检索系统。
4. **动态性原则**：系统随时间变化发展。应用：要求信息资源建设适应社会信息需求变化，关注趋势、用户需求和外部环境。

**解析**：本题是高频简答题。需分别阐述四大原则的内涵和应用，注意联系性原则中的"共建共享"是核心考点。

</details>

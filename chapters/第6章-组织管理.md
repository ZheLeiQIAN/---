## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **复选（Re-selection）** | 在藏书补充过程后一阶段，对馆内入藏书刊的再次选择；与初选构成辩证统一过程 | 苏联格利科尔耶夫60年代提出，斯多利亚洛夫1979系统阐述 |
| **剔除（Weeding/Deselection）** | 根据一定原则和标准，将长期滞留在书架上、利用率极低或已失去利用价值的文献从馆藏体系中分离出来，并按不同情况分别处理 | 复选的结果性操作 |
| **滞架时限（Shelf-stand Time）** | 文献在两次流通之间（含最后一次流通至今）滞留在书架上未被使用的时间长度 | 确定流通书库中文献馆内调配和去留的**最佳标准** |
| **半衰期（Half-life）** | 某学科被利用的文献总量中，一半文献失去利用效率所经历的时间 | 1960年伯顿和凯普勒提出 |
| **分类排架法** | 按文献本身内容所属的学科体系来排列藏书的方法；由分类号和辅助号组成分类排架号 | **主要的**馆藏文献排列方法 |
| **贮存图书馆（Depository Library）** | 为解决收藏空间紧张、节省管理成本、提高利用率而设立的专门收藏利用率相对较低文献资源的图书馆 | txt要求3种模式 |
| **NDPP** | 国家数字科技文献资源长期保存体系（National Digital Preservation Program），我国科技文献长期保存机构 | 保障科技文献长期可持续访问 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **初选（Initial Selection）** | 补充过程开始阶段对馆外出版物的入藏选择 | 与复选构成辩证统一过程 |
| **馆藏文献布局** | 将图书馆入藏的全部文献，按一定标准划分为若干部分，建立各种功能的书库、阅览空间 | 实质是空间位置上的划分 |
| **塔式垂直布局** | 按图书馆建筑结构，以垂直方向层层叠置的布局方式 | 高层书库，节省占地 |
| **水平展开布局** | 按图书馆建筑结构，以水平方向展开的布局方式 | 低层大面积，便于阅览 |
| **主题排架法** | 按文献主题排列藏书的方法 | 按内容排架的一种 |
| **固定排架法** | 按出版物的固定编号排架；编号由库室号、书架号、层格号、书位号组成 | 按形式排架，位置固定 |
| **登记号排架法** | 按文献登记顺序排列的排架方法 | 按形式排架，一书一号 |
| **色标排架法** | 以颜色标识区分文献的排架方法 | 按形式排架，辅助性视觉标识 |
| **调拨** | 主管部门组织，有计划地将馆藏文献调拨给其他图书馆 | 处置方式之一 |
| **置换** | 以本馆多余文献与其他图书馆交换所需文献 | 处置方式之一 |

## 【方法比较】

### 1. 复选标准9种比较 ★★★

| 标准 | 核心判断依据 | 关键要点 |
|------|------------|----------|
| **内容标准** | 文献内容是否陈旧过时、重复、不符合馆藏政策 | 包括内容不适于公开流通 |
| **外形标准** | 文献外观是否陈旧、污损、缺页、破损 | 视听/缩微/电子文献含制作质量与读取能力 |
| **书龄标准** | 出版/版权/印刷/采购/进入流通日期至复选时的年限 | 以时间年限衡量 |
| **滞架时限标准** | 两次流通间滞留书架未被使用的时间长度 | **最佳标准**（确定流通书库文献去留） |
| **使用频率标准** | 一定时期内流通、阅览、检索或下载次数 | 低于规定次数为复选对象 |
| **复本量标准** | 复本量多寡是否符合读者需求 | 多余复本+复本不足两种情况 |
| **替代标准** | 不同类型文献间的相互替代性 | 主要考虑电子文献对印刷型文献的替代 |
| **语种标准** | 馆藏文献的语种 | 语种与读者掌握语种不一致、无人问津的语种 |
| **保障标准** | 文献的保障要求 | 馆际协作/馆藏体系明确要求保障的不复选或仅剔复本 |

### 2. 复选方法9种比较 ★★★

| 方法 | 操作原理 | 优缺点/前提条件 |
|------|---------|----------------|
| **经验判断法** | 根据文献外观、价值、读者取向、借阅频率等感性认识和经验决定取舍 | 简单但缺乏精确，主观差异大 |
| **滞架时限法** | 根据文献两次流通间滞留未被使用的时间长度确定去留 | 确定"呆滞书刊"的有效方法，前提是完整借阅记录 |
| **书龄法** | 根据书龄长短确定取舍 | 依据半衰期理论，分析不同书龄馆藏结构和使用状况 |
| **半衰期测定法** | 根据学科文献半衰期规律确定各类藏书取舍 | 需先确定各学科文献不同有效时间 |
| **目录比较法** | 将馆藏目录与推荐书目、权威书目比较进行复选 | 如比较馆藏期刊目录与核心期刊目录 |
| **用户评议法** | 通过访问、交谈或表格调查了解用户评价与建议 | 反映读者需求，但受用户主观影响 |
| **内容层次判断法** | 将文献按内容深浅分层次，低于所需层次的列为剔除对象 | 适用于层次分明的馆藏体系 |
| **数学计算方法** | 用数量表示变量，运用公式计算后做出复选决定 | 载文量、引文量、书龄、复本量、滞架时限、使用频率为重要参数 |
| **外形判断方法** | 以文献外形为剔除标准 | 多次修补无法使用、质量差影响阅读、污损缺页严重者剔除 |

### 3. 复选标准 vs 复选方法（区分）★

| 比较维度 | 复选标准（9种） | 复选方法（9种） |
|----------|---------------|---------------|
| **性质** | 判断依据 | 操作手段 |
| **回答的问题** | "依据什么判断文献去留" | "如何具体操作复选" |
| **举例** | 滞架时限标准（依据什么） | 滞架时限法（如何操作） |
| **关系** | 标准是方法的基础 | 方法是标准的操作化 |

**注意**：复选标准与复选方法不是简单的一一对应关系。

### 4. 馆藏排架方式对比

| 分类 | 排架法 | 排列依据 | 特点 |
|------|--------|---------|------|
| **按内容** | **分类排架法** ★ | 学科体系（分类号+辅助号） | **主要的**排列方法；同类书集中，便于按类检索 |
| **按内容** | 主题排架法 | 文献主题 | 按主题集中相关文献 |
| **按形式** | 登记号排架法 | 个别登记顺序 | 一书一号，简单清楚，不反映内容 |
| **按形式** | 固定排架法 | 固定编号（库室号+书架号+层格号+书位号） | 位置固定，不倒架 |
| **按形式** | 色标排架法 | 颜色标识 | 辅助性视觉标识 |

### 5. 馆藏布局两种方式对比

| 比较维度 | 塔式垂直布局 | 水平展开布局 |
|----------|-----------|-----------|
| **结构方向** | 垂直方向层层叠置 | 水平方向展开 |
| **建筑特点** | 高层书库 | 低层大面积 |
| **空间利用** | 节省占地面积 | 占地面积大 |
| **阅览特点** | 各层独立，垂直交通为主 | 平面展开，便于阅览流通 |
| **适用场景** | 用地紧张的城市图书馆 | 用地充裕的郊区或校园图书馆 |

### 6. 贮存图书馆3种模式对比（txt版本）★

| 模式 | 建立主体 | 服务对象 | 特点 |
|------|---------|---------|------|
| **专有贮存** | 各图书馆自行建立 | 本馆利用率较低的书刊资料 | 独立性强，服务本馆 |
| **地区合作贮存** | 一定地区内若干图书馆联合建立 | 本地区各参加馆剔除的罕用文献 | 合作共享，服务地区 |
| **国家贮存** | 国家建立 | 各类型图书馆送交的利用率较低文献 | 国家层面，服务全国 |

### 7. 评价方法6种比较

| 方法 | 原理 | 适用场景 | 局限性 |
|------|------|---------|--------|
| **专家评价法** | 学者专家直接观察馆藏评价 | 了解馆藏范围、规模、深度 | 主观性强 |
| **用户评议法** | 向用户征求意见 | 了解读者需求和馆藏利用效能 | 受用户范围和参与度影响 |
| **统计分析法** | 运用馆藏统计数据分析和评价 | 运用广泛且有效 | 需完整统计数据支撑 |
| **书目核对法** | 与标准书目、核心书目逐一核对 | 评价馆藏文献质量 | 书目有限不能全面反映 |
| **引文分析法** | 查对学术论著参考文献被本馆收藏情况 | 反映馆藏被利用和可能被利用情况 | 需有代表性读者研究成果 |
| **成本收益分析法** | 对数字资源成本与收益比较 | 评价数字资源质量及经费分配合理性 | 成本和收益量化难度大 |

2. **外形标准**：以文献外观是否陈旧妨碍使用、纸张印刷装订质量、污损缺页、多次修补无法使用等为标准

7. **替代标准**：以不同类型文献间的相互替代性为标准，主要考虑电子文献对印刷型文献的替代性

9. **保障标准**：以文献的保障要求为标准

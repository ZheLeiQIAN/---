## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **文献采选3依据** | 文献资源采选的三个基本依据：馆藏政策、读者需求、经费条件 | 与电子资源4依据对比，电子多出"资源品质""定价模式"两项 |
| **馆藏政策** | 图书馆根据其性质、任务和读者需求制定的文献资源建设指导性文件 | 采选的根本依据，文献与电子采选共有 |
| **读者需求** | 读者对文献信息的实际需要 | 文献采选的直接依据；电子采选称"用户需求" |
| **经费条件** | 图书馆可用于文献采选的经费预算及财务状况 | 文献采选特有依据，电子采选以"定价模式"替代 |
| **五级等级结构** | 完整级→研究级→大学级→基础级→最低级，由高到低的藏书建设等级体系 | 五级递降，完整级最高、最低级在范围之外 |
| **完整级** | 完整齐全搜集某专题领域所有知识记录为目标的藏书方案 | 最高级别，国家图书馆核心领域适用 |
| **研究级** | 以满足独立研究需要为目标的藏书方案 | 特藏应达到研究级 |
| **大学级** | 以满足大学生和个人自学大学课程需要为目标的藏书方案 | 高校图书馆基础配置 |
| **基础级** | 以介绍人们认识不同专业领域为目标的藏书方案 | 普及性，县级馆适用 |
| **最低级** | 在藏书范围之外的藏书方案 | 非馆藏重点，最低层次 |
| **纲目订书** | 图书馆将藏书补充计划按入藏文献范围、内容、出版社、价格等列成纲目提交给书商，书商按"纲目"直接寄书 | 退书率≤10%，签一次合同连续供书 |
| **长期订购** | 保证多卷和连续出版物完整入藏的手段，一次手续长期有效 | 可享10%-30%优惠 |
| **集中式交换** | 将可供交换文献送交交换中心，由中心集中交换 | 最基本最常用，非营利，仅收少许手续费 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **全面采选** | 中文各学科各类型文献全面入藏 | 国家图书馆级别一 |
| **着重采选** | 古籍善本、名人手稿、珍稀文献 | 国家图书馆级别二 |
| **适当采选** | 外文文献适时调整范围和重点 | 国家图书馆级别三 |
| **不宜采选** | 范围之外文献 | 国家图书馆级别四 |
| **双边式交换** | 双方直接互寄文献 | 涉及范围广、周期长 |
| **多边式交换** | 各馆送书目到交换书目中心，中心分发供求书目 | 交换对象不确定 |
| **提类订书** | 按类别集中订购 | 按学科类别批量采购 |
| **全数订购** | 某出版社文献全部订购 | 保证完整性 |
| **集中订购** | 集中向某一供应商订购 | 统一采购降低成本 |
| **租借订购** | 先租后买，试用满意再正式采购 | 降低采购风险 |

## 【重点公式/算法】

### 1. 文献资源采选3依据 ★★

```
文献资源采选3依据
│
├─ ① 馆藏政策（根本依据）
│      └─ 图书馆性质、任务、读者需求制定的指导性文件
│
├─ ② 读者需求（直接依据）
│      └─ 读者对文献信息的实际需要，决定采选范围、深度、重点
│
└─ ③ 经费条件（现实约束）
       └─ 经费预算决定采选品种和数量
```

**对比关系**：文献3依据 vs 电子4依据（详见Ch5）

```
                文献采选3依据              电子采选4依据（Ch5）
馆藏政策         ✓ 共有                    ✓ 共有
需求             ✓ 读者需求                 ✓ 用户需求
品质/定价        ✗ 无                      ✓ 资源品质 + 定价模式（电子特有）
经费             ✓ 经费条件（文献特有）     ✗ 以定价模式替代
```

**记忆要点**：文献重"经费"，电子重"品质+定价"；两者都含"馆藏政策"和"需求"。

### 2. 五级等级结构（由高到低）★★★

```
完整级  ★★★  ── 完整齐全搜集某专题领域所有知识记录
  │           （国家图书馆核心领域）
  ▼
研究级  ★★   ── 满足独立研究需要
  │           （特藏、重点学科应达到此级）
  ▼
大学级  ★    ── 满足大学生和个人自学大学课程需要
  │           （高校图书馆基础配置）
  ▼
基础级       ── 介绍人们认识不同专业领域

              （非馆藏重点）
```

**记忆口诀**：完研大基最，五级从高降；国家求完整，特藏达研究。

```
采选四级别：
```

### 5. 捐赠及注意事项（6项）★

```
非购买方式：捐赠
│
├─ ① 捐赠协议 ── 与捐赠方签订协议，明确捐赠内容、数量、用途等
├─ ② 质量把关 ── 捐赠文献应符合馆藏建设方针，不符合者可婉拒或转赠
├─ ③ 产权归属 ── 明确捐赠文献产权归属图书馆，避免后续纠纷
├─ ④ 登记入藏 ── 按正常流程登记入藏，纳入馆藏体系
├─ ⑤ 特殊要求 ── 捐赠方有特殊要求的（如专架陈列、不得外借等），应在协议中明确
└─ ⑥ 感谢回执 ── 向捐赠方出具感谢信或接收凭证，体现尊重和规范
```

**记忆口诀**：协议质量产权登记，特殊感谢六项齐。

### 6. 交换三种形式

```
（1）双边式交换
    A ◄────直接寄送────► B
    范围广、周期长、不确定性高

（2）多边式交换
    各馆 ──可供文献/需求──► 交换书目中心 ──分发供求书目──► 各馆
    麻烦、交换对象不确定、跨国费用高

（3）集中式交换 ★（最基本最常用）
    各馆 ──可供文献──► 文献交换中心 ──集中交换──► 各馆
    简单易行、周期短、针对性强、非营利、仅收少许手续费
```

### 7. 文献采集四阶段流程

```
选书阶段 ──────► 订购阶段 ──────► 验收阶段 ──────► 补缺阶段
   │                │                │                │
   ▼                ▼                ▼                ▼
搜集书目信息    采购行动实施    财产验收比对      催缺未到图书
审查圈选        办理审批手续    实体核实          催缺仍不到馆
查重            填报预订单      财产分配/登记     撤销原有订单
                寄发订购单      经费统计支出      其他渠道重订
```

### 8. 选择馆配商5种方式

```
（1）公开招标      ← 中文图书
（2）邀请招标
（3）竞争性谈判    ← 外文图书、外文期刊
```

## 【方法比较】

### 1. 文献采选3依据 vs 电子采选4依据 ★★

| 比较维度 | 文献采选3依据（Ch4） | 电子采选4依据（Ch5） |
|----------|-------------------|-------------------|
| **依据数量** | 3个 | 4个 |
| **共有依据** | 馆藏政策、需求（读者需求） | 馆藏政策、需求（用户需求） |
| **特有依据** | 经费条件 | 资源品质、定价模式 |
| **关注重点** | 经费约束、读者覆盖 | 品质评价、定价策略 |
| **差异原因** | 印本文献定价相对简单，主要受经费约束 | 电子资源定价模式复杂（3大类11种），需专门评价品质与价格 |

**记忆逻辑**：文献3依据 = 共有2（馆藏政策+需求） + 经费条件；电子4依据 = 共有2 + 资源品质 + 定价模式。

### 2. 五级等级结构详解 ★★★

| 级别 | 名称 | 目标 | 适用馆型 |
|------|------|------|----------|
| **第一级** | 完整级 | 完整齐全搜集某专题领域所有知识记录 | 国家图书馆核心领域 |
| **第二级** | 研究级 | 满足独立研究需要 | 特藏、重点学科 |
| **第三级** | 大学级 | 满足大学生和个人自学大学课程需要 | 高校图书馆基础配置 |
| **第四级** | 基础级 | 介绍人们认识不同专业领域 | 普及性，县级馆适用 |
| **第五级** | 最低级 | 在藏书范围之外 | 非馆藏重点 |

### 3. 交换三种形式对比 ★★

| 维度 | 双边式交换 | 多边式交换 | 集中式交换 ★ |
|------|-----------|-----------|-------------|
| **方式** | 双方直接互寄文献 | 各馆送书目到交换书目中心，中心分发供求书目 | 各馆送文献到交换中心，中心集中交换 |
| **特点** | 涉及范围广 | 交换对象不确定 | 简单易行 |
| **周期** | 较长 | 较麻烦 | 短 |
| **针对性** | 不确定 | 不确定 | 强 |
| **地位** | 基本方式 | 较少使用 | 最基本最常用 |
| **费用** | 邮寄直接 | 跨国需邮寄两次以上，费用增加 | 非营利机构，仅收少许手续费 |

- 有出版物交存制度保证
- 经费充裕
- 重视书刊交换

### 题目3（简答题）

**简述文献交换的三种形式及其特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献交换主要有三种形式：

1. **双边式交换**：双方直接互寄文献。涉及范围广，但周期长，不确定性高，是基本方式。

2. **多边式交换**：各馆将可供交换文献和需求送至交换书目中心，由中心分发供求书目。较为麻烦，交换对象不确定，跨国费用较高，较少使用。

3. **集中式交换**：各馆将可供交换文献送交文献交换中心，由中心集中交换。简单易行、周期短、针对性强，是非营利机构，仅收少许手续费，是最基本最常用的方式。

**解析**：本题是高频简答题。关键在于区分三种形式的方式和特点，特别要注意集中式交换是"最基本最常用"的方式。

</details>

### 题目5（简答题）

<details>

**答案：**

**一、文献资源采选3依据**

3. **经费条件**：图书馆可用于文献采选的经费预算及财务状况，是采选的现实约束，决定采选的品种和数量。

电子资源采选4依据为：馆藏政策、用户需求、资源品质、定价模式（详见第5章）。

|------|----------|----------|
| 共有 | 馆藏政策、需求（读者） | 馆藏政策、需求（用户） |
| 特有 | 经费条件 | 资源品质、定价模式 |

差异原因：印本文献定价相对简单，主要受经费约束；电子资源定价模式复杂（3大类11种），需专门评价品质与价格。

**解析**：本题考查文献采选依据及与电子采选依据的对比。注意两者共有"馆藏政策"和"需求"，文献特有"经费条件"，电子特有"资源品质"和"定价模式"。

### 题目7（简答题）

**简述文献采集的四个阶段。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献采集包括四个阶段：

1. **选书阶段**：搜集书目信息→审查圈选→查重，确定待采选文献。
2. **订购阶段**：采购行动实施→办理审批手续→填报预订单→寄发订购单。
3. **验收阶段**：实体核实→财产分配→登记→经费统计支出。
4. **补缺阶段**：催缺未到图书→催缺仍不到馆→撤销原有订单→其他渠道重订。

**解析**：本题考查采选流程。四个阶段顺序固定：选书→订购→验收→补缺。

</details>

### 题目8（简答题）

**简述图书馆接受文献捐赠时需要注意的6个问题。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

图书馆接受文献捐赠时需要注意以下6个问题：

1. **捐赠协议**：与捐赠方签订正式协议，明确捐赠文献的内容、数量、用途等，确保双方权利义务清晰。
2. **质量把关**：捐赠文献应符合本馆藏书建设方针和收藏范围，不符合要求的文献可婉拒或转赠他馆。
3. **产权归属**：明确捐赠文献的产权归属图书馆，避免后续处置时产生纠纷。
4. **登记入藏**：捐赠文献应按正常采选流程登记入藏，纳入馆藏体系统一管理。
5. **特殊要求**：捐赠方如有特殊要求（如专架陈列、不得外借、永久保存等），应在捐赠协议中明确约定。
6. **感谢回执**：向捐赠方出具感谢信或正式接收凭证，体现对捐赠行为的尊重和管理的规范。

**解析**：本题考查捐赠注意事项。记忆口诀："协议质量产权登记，特殊感谢六项齐。" 注意捐赠是非购买方式获取文献的重要途径，与交换同属非购买方式，但捐赠需特别注意产权归属和质量把关。

</details>

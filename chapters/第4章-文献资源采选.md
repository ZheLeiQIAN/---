## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **五级等级结构** | 完整级→研究级→大学级→基础级→最低级，由高到低的藏书建设等级体系 | 五级递降，完整级最高、最低级在范围之外 |
| **完整级** | 完整齐全搜集某专题领域所有知识记录为目标的藏书方案 | 最高级别，国家图书馆核心领域适用 |
| **研究级** | 以满足独立研究需要为目标的藏书方案 | 特藏应达到研究级 |
| **大学级** | 以满足大学生和个人自学大学课程需要为目标的藏书方案 | 高校图书馆基础配置 |
| **基础级** | 以介绍人们认识不同专业领域为目标的藏书方案 | 普及性，县级馆适用 |
| **最低级** | 在藏书范围之外的藏书方案 | 非馆藏重点，最低层次 |
| **四级数字资源结构** | 永久保存级→服务级→镜像级→链接级，按数字资源重要程度和保存方式分为四个级别 | 唯一性递减，稳定性递减 |
| **永久保存级馆藏** | 保存价值已鉴定、用途得到认可、具有唯一性，且仅定位在某特定图书馆，其他地方不可获得 | 最高级别，唯一性是核心 |
| **服务级馆藏** | 对用户有用且必须用的虚拟馆藏；本馆用户十分需要，但贮存在其他数字图书馆服务器上 | 虚拟馆藏，存于他处 |
| **镜像级馆藏** | 其他数字图书馆馆藏的拷贝，地址可能移动，经常不可存取 | 拷贝性质，不稳定 |
| **链接级馆藏** | 在任何地方均可寄居的信息资料 | 最低级别 |
| **纲目订书** | 图书馆将藏书补充计划按入藏文献范围、内容、出版社、价格等列成纲目提交给书商，书商按"纲目"直接寄书 | 退书率≤10%，签一次合同连续供书 |
| **长期订购** | 保证多卷和连续出版物完整入藏的手段，一次手续长期有效 | 可享10%-30%优惠 |
| **集中式交换** | 将可供交换文献送交交换中心，由中心集中交换 | 最基本最常用，非营利，仅收少许手续费 |
| **国家采购** | 对与国家经济及社会发展战略目标相关的、教学科研人员和社会公众普遍需要的基础性、综合性数字资源，利用公共财政即政府出资购买的采购模式 | 3型：统购/补贴/平台 |
| **集团采购** | 若干图书馆自愿组成集团，共同推举谈判代表与电子资源提供商谈判价格与使用条款 | DRAA是典型代表 |
| **DRAA** | 高校图书馆数字资源采购联盟（Digital Resource Acquisition Alliance of Chinese Academic Libraries） | 中国高校集团采购核心组织 |
| **单馆自主采购** | 各馆根据本馆需要，自发与信息资源提供商谈判价格并自行签署购买使用协议 | 适用于价格不高而利用率较高的数据库 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **国家书目** | 系统收录一个国家一定时期内全部出版物的书目 | 属于参考性书目 |
| **全面采选** | 中文各学科各类型文献全面入藏 | 国家图书馆级别一 |
| **着重采选** | 古籍善本、名人手稿、珍稀文献 | 国家图书馆级别二 |
| **适当采选** | 外文文献适时调整范围和重点 | 国家图书馆级别三 |
| **不宜采选** | 范围之外文献 | 国家图书馆级别四 |
| **双边式交换** | 双方直接互寄文献 | 涉及范围广、周期长 |
| **多边式交换** | 各馆送书目到交换书目中心，中心分发供求书目 | 交换对象不确定 |
| **统购型** | 国家利用中央经费统一购买电子资源，全国每台联网计算机均可免费访问 | 中央经费全额，如冰岛 |
| **补贴型** | 政府支付一部分，另一部分由各图书馆支付 | 政府+各馆分担，如加拿大 |
| **平台型** | 国家投资建立统一采购与访问平台，平台费由国家支付，资源购买费由基层馆负担 | 国家出平台费+基层出资源费，如英国 |

## 【重点公式/算法】

### 1. 五级等级结构（由高到低）★★★

```
完整级  ★★★  ── 完整齐全搜集某专题领域所有知识记录
  │           （国家图书馆核心领域）
  ▼
研究级  ★★   ── 满足独立研究需要
  │           （特藏、重点学科应达到此级）
  ▼
大学级  ★    ── 满足大学生和个人自学大学课程需要
  │           （高校图书馆基础配置）
  ▼
基础级       ── 介绍人们认识不同专业领域
  │           （普及性，县级馆适用）
  ▼
最低级       ── 在藏书范围之外
              （非馆藏重点）
```

**记忆口诀**：完研大基最，五级从高降；国家求完整，特藏达研究。

### 2. 四级数字资源结构

```
永久保存级 ──→ 服务级 ──→ 镜像级 ──→ 链接级
（最高）                                        （最低）
唯一性          虚拟馆藏       拷贝性质       寄居信息
不可获得        存于他处       不稳定         任何地方
```

**记忆口诀**：永服镜链，从高到低；永久唯一，链接最弱。

### 3. 国家图书馆采选方针与四级别

```
采选方针："中文求全"、"国内出版物求全"与"外文求精"

采选四级别：
├─ 全面采选   ← 中文各学科各类型文献全面入藏
├─ 着重采选   ← 古籍善本、名人手稿、珍稀文献
├─ 适当采选   ← 外文文献适时调整范围和重点
└─ 不宜采选   ← 范围之外文献
```

### 5. 交换三种形式

```
（1）双边式交换
    A ◄────直接寄送────► B
    范围广、周期长、不确定性高

（2）多边式交换
    各馆 ──可供文献/需求──► 交换书目中心 ──分发供求书目──► 各馆
    麻烦、交换对象不确定、跨国费用高

（3）集中式交换 ★（最基本最常用）
    各馆 ──可供文献──► 文献交换中心 ──集中交换──► 各馆
    简单易行、周期短、针对性强、非营利、仅收少许手续费
```

## 【方法比较】

### 1. 五级等级结构详解 ★★★

| 级别 | 名称 | 目标 | 适用馆型 |
|------|------|------|----------|
| **第一级** | 完整级 | 完整齐全搜集某专题领域所有知识记录 | 国家图书馆核心领域 |
| **第二级** | 研究级 | 满足独立研究需要 | 特藏、重点学科 |
| **第三级** | 大学级 | 满足大学生和个人自学大学课程需要 | 高校图书馆基础配置 |
| **第四级** | 基础级 | 介绍人们认识不同专业领域 | 普及性，县级馆适用 |
| **第五级** | 最低级 | 在藏书范围之外 | 非馆藏重点 |

### 2. 四级数字资源结构对比 ★★

| 级别 | 名称 | 核心特征 | 获取方式 | 稳定性 |
|------|------|----------|----------|--------|
| **第一级** | 永久保存级馆藏 | 保存价值已鉴定、用途得到认可、具有唯一性，仅定位在某特定图书馆 | 本馆永久保存 | 最高 |
| **第二级** | 服务级馆藏 | 对用户有用且必须用的虚拟馆藏，贮存在其他数字图书馆服务器上 | 远程访问 | 较高 |
| **第三级** | 镜像级馆藏 | 其他数字图书馆馆藏的拷贝，地址可能移动，经常不可存取 | 本地镜像拷贝 | 较低 |
| **第四级** | 链接级馆藏 | 在任何地方均可寄居的信息资料 | 链接访问 | 最低 |

**记忆逻辑**：永久保存（唯一/不可替代）> 服务（需要但存于他处）> 镜像（拷贝但不稳定）> 链接（寄居/最弱）。

### 3. 交换三种形式对比 ★★

| 维度 | 双边式交换 | 多边式交换 | 集中式交换 ★ |
|------|-----------|-----------|-------------|
| **方式** | 双方直接互寄文献 | 各馆送书目到交换书目中心，中心分发供求书目 | 各馆送文献到交换中心，中心集中交换 |
| **特点** | 涉及范围广 | 交换对象不确定 | 简单易行 |
| **周期** | 较长 | 较麻烦 | 短 |
| **针对性** | 不确定 | 不确定 | 强 |
| **地位** | 基本方式 | 较少使用 | 最基本最常用 |
| **费用** | 邮寄直接 | 跨国需邮寄两次以上，费用增加 | 非营利机构，仅收少许手续费 |

### 4. 三种电子资源采购方式对比 ★★★

| 比较维度 | 国家采购 | 集团采购 | 单馆自主采购 |
|----------|----------|----------|--------------|
| **含义** | 政府出资购买基础性、综合性数字资源 | 若干图书馆自愿组成集团共同谈判 | 各馆自发与提供商谈判签约 |
| **资金来源** | 公共财政（政府出资） | 各成员馆自行支付 | 各馆自行支付 |
| **优势** | 缓解数字鸿沟；提供平等访问权；统一市场获价格优惠；有效利用政府财政 | 享受批量购买优惠；节省谈判时间和人力成本；加强馆际交流与合作 | 针对性强采集本馆所需资源；便于形成本馆特色资源体系；可定制个性化服务 |
| **不足** | 采选针对性不强；削弱基层图书馆自主选择权 | 成员馆分散，联系沟通困难；系统风险 | 资源采集容易盲目；不利于应对数据库商提价；长期保存代价较高 |
| **适用类型** | 综合性和基础性的公共基础信息 | 综合性、多学科全文数据库 | 专业性、边缘性学科数据库 |
| **对应层次** | 公共基础层信息 | 综合性、多学科全文数据库层 | 专业性、边缘性学科数据库层 |
| **典型实例** | 国家自然科学基金委和国家科技图书文献中心 | DRAA、CALIS、CSDL | CNKI系列数据库、万方数据 |

**演变逻辑**：国家采购（宏观基础保障）→ 集团采购（中观协同议价）→ 单馆采购（微观特色补充），三者互补而非替代。

### 5. 国家采购三类型对比

| 类型 | 具体方式 | 经费来源 | 典型国家 |
|------|----------|----------|----------|
| **统购型** | 国家利用中央经费统一购买电子资源，全国每一台联网计算机均可免费访问 | 中央经费全额 | 冰岛 |
| **补贴型** | 政府支付一部分电子资源费用，另一部分由各个图书馆支付 | 政府+各馆分担 | 加拿大 |
| **平台型** | 国家投资建立统一的电子资源采购与访问平台，平台运转经费由国家支付，但资源购买费由基层图书馆负担 | 国家出平台费+基层出资源费 | 英国 |

### 6. 大宗采购6种方式对比

| 方式 | 核心特点 | 关键要点 |
|------|---------|----------|
| **纲目订书** | 列纲目交书商，直接寄书 | 退书率≤10%，连续供书 |
| **提类订书** | 按类别集中订购 | 按学科类别批量采购 |
| **全数订购** | 某出版社文献全部订购 | 保证完整性 |
| **集中订购** | 集中向某一供应商订购 | 统一采购降低成本 |
| **租借订购** | 先租后买，试用满意再正式采购 | 降低采购风险 |
| **长期订购** | 一次手续长期有效 | 保证多卷和连续出版物完整入藏，优惠10%-30% |

### 7. 捐赠注意事项

| 注意事项 | 说明 |
|---------|------|
| **捐赠协议** | 应与捐赠方签订捐赠协议，明确捐赠内容、数量、用途等 |
| **质量把关** | 捐赠文献应符合馆藏建设方针，不符合者可婉拒或转赠 |

### 题目2（名词解释）

**请解释"四级数字资源结构"的内涵，并说明各级馆藏的特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**四级数字资源结构**是电子馆藏体系等级结构的一种分类方式，按照数字资源的重要程度和保存方式分为四个级别：

1. **永久保存级馆藏**：保存价值已得到鉴定；用途得到认可；具有唯一性；并仅仅定位在某一个特定的图书馆，其他地方不可获得。最高级别。
2. **服务级馆藏**：对用户有用且必须用的虚拟馆藏。本馆用户十分需要，但贮存在其他数字图书馆的服务器上。
3. **镜像级馆藏**：其他数字图书馆馆藏的拷贝，其地址可能会慢慢移动，并经常不可存取；在其他地方也很容易找到该资料。
4. **链接级馆藏**：在任何地方均可寄居的信息资料，最低级别。

**解析**：本题是高频名词解释题。注意"永久保存级"的"唯一性"和"不可获得性"是核心关键词，"服务级"是"虚拟馆藏"且"存于他处"，"镜像级"是"拷贝"且"不稳定"，"链接级"是"寄居"且最弱。

</details>

### 题目3（简答题）

**简述国家图书馆文献采选的特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

国家图书馆文献采选的特点主要体现在采选方针和采选级别两个方面：

**一、采选方针**
- **"中文求全"**：国内出版物全面系统收集保存
- **"外文求精"**：外文文献精选入藏

**二、采选四级别**
1. **全面采选**：中文各学科各类型文献全面入藏

**三、其他特点**
- 任务重、数量大、范围广、种类全
- 有出版物交存制度保证
- 经费充裕
- 重视书刊交换

**解析**：本题考查国家图书馆采选特点。核心是"中文求全、外文求精"方针和四个采选级别。

### 题目4（简答题）

**简述文献交换的三种形式及其特点。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献交换主要有三种形式：

1. **双边式交换**：双方直接互寄文献。涉及范围广，但周期长，不确定性高，是基本方式。

2. **多边式交换**：各馆将可供交换文献和需求送至交换书目中心，由中心分发供求书目。较为麻烦，交换对象不确定，跨国费用较高，较少使用。

3. **集中式交换**：各馆将可供交换文献送交文献交换中心，由中心集中交换。简单易行、周期短、针对性强，是非营利机构，仅收少许手续费，是最基本最常用的方式。

**解析**：本题是高频简答题。关键在于区分三种形式的方式和特点，特别要注意集中式交换是"最基本最常用"的方式。

</details>

### 题目5（简答题）

**简述电子资源三种采购方式及其比较。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

电子资源采购方式主要有三种：

1. **国家采购**：对与国家经济及社会发展战略目标相关的、教学科研人员和社会公众普遍需要的基础性、综合性数字资源，利用公共财政（政府出资）购买的采购模式。分为统购型、补贴型、平台型三种类型。适用于综合性和基础性的公共基础信息。

2. **集团采购**：若干图书馆自愿组成集团，共同推举谈判代表与电子资源提供商谈判价格与使用条款，最终购买合同由提供商与各加盟馆签订，购买费用由各成员馆自行支付。DRAA（高校图书馆数字资源采购联盟）是典型代表。适用于综合性、多学科全文数据库。

3. **单馆自主采购**：各馆根据本馆需要，自发与信息资源提供商谈判价格并自行签署购买使用协议。适用于价格不高而利用率较高的数据库，主要是专业性、边缘性学科数据库。

**演变逻辑**：国家采购（宏观基础保障）→ 集团采购（中观协同议价）→ 单馆采购（微观特色补充），三者互补而非替代。

**解析**：本题是高频简答与论述题。需准确记忆三种方式的含义、资金来源、适用类型和典型实例，并理解三者间的互补关系。

</details>

## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **信息生命周期** | 信息从生成到失效的全过程：生成→采集→存储/组织/加工→利用→维护→失效/剔除 | 信息资源建设就是完成一个生命周期的过程 |
| **布拉德福定律** | 将期刊按载文量分为三区，每区文献数约为总数的1/3，期刊数成等比数列（公比≈5） | P1:P2:P3 = 1:n:n² |
| **半衰期（Half-life）** | 某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的 | 文献老化理论的标志概念 |
| **文献老化** | 文献内容的老化和文献载体的老化 | 内容老化是研究重点 |
| **价值论选书观** | 杜威、海恩斯主张选择科学或艺术价值高的出版物，以文献内容价值为选择依据 | 片面性：不考虑读者需求 |
| **需要论选书观** | 普尔·克特强调藏书补充的实用性，以用户需要为依据 | 片面性：忽视社会阅读导向 |
| **使用概率论** | 阮冈纳赞提出入藏每本书是否被利用是随机事件，概率越大表示被利用可能性越大 | 引入数学方法，客观但操作难 |
| **成本-收益理论** | 通过调控手段协调信息资源在时间、空间、部门、数量上的分布，使有限资源产生最大效益 | 经济学命题 |
| **开放获取（OA）** | 科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权 | 保留著作权+免费使用 |
| **金色OA** | 开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取 | 开放期刊+开放论文；可能有APC |
| **绿色OA** | 开放存储（Open Access Archives），将研究成果存储到机构或学科知识库 | 自存储；可能有 embargo 时滞 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **普赖斯曲线** | 文献增长与时间成指数函数关系：F(t) = ae^(bt) | 指数增长模型 |
| **综合计分法** | 麦高文从文献知识价值和社会需求两方面综合评价 | 辩证但难实行 |
| **社会调查选择论** | 维拉德依据对读者的社会、心理研究结果进行选书 | 按年龄、教育、职业分组 |
| **会计成本** | 实际发生的成本 | 成本类型之一 |
| **机会成本** | 选择某种东西而放弃的其他东西的成本 | 成本类型之一 |

## 【重点公式/算法】

### 1. 布拉德福定律公式 ★★★

```
P1 : P2 : P3 = 1 : n : n²

其中：
- P1 = 核心区期刊数量
- P2 = 相关区期刊数量
- P3 = 相邻区期刊数量
- n ≈ 5（公比近似为5）
```

**含义**：将某学科期刊按载文量降序排列，分为三个区，每区载文量约为该学科总文献数的1/3，但三个区的期刊数量之比约为 1 : 5 : 25。

**对信息资源建设的意义**：
1. 确定本学科核心期刊，以有限经费获得高质量、使用率高的文献
2. 确定合理的藏书规模，规划馆藏文献布局

### 2. 半衰期概念

```
半衰期 = 某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的

特征：
- 反映文献老化速度
- 不同学科半衰期不同（基础学科长、应用学科短）
- 为评价选择、优化结构、制定原则提供依据
```

**辨识要点**：文献半衰期反映的是**文献老化**的特征，属于文献老化理论的核心概念。

### 3. 普赖斯曲线（文献增长公式）

```
F(t) = a × e^(bt)

其中：
- F(t) = 时刻t的文献累积量
- a = 统计初始时刻的文献量
- e = 2.718（自然对数底）
- b = 持续增长率
- t = 时间
```

### 4. 信息生命周期流程

```
生成 → 采集 → 存储/组织/加工 → 利用 → 维护 → 失效/剔除
```

**核心结论**：信息资源建设就是完成一个信息生命周期的过程。

### 5. 成本-收益理论四维调控

```
四维调控：时间 · 空间 · 部门 · 数量

成本类型：
├─ 会计成本：实际发生的成本（购书经费、设备费用、人员工资）
└─ 机会成本：选择某种方案而放弃的其他方案的收益
```

## 【方法比较】

### 1. 五大选书理论比较 ★★★

| 选书理论 | 代表人物 | 核心观点 | 优点 | 片面性/局限 |
|----------|----------|----------|------|-------------|
| **价值论** | 杜威、海恩斯 | 选择科学或艺术价值高的出版物 | 重视文献质量 | 只重价值，不考虑读者实际需求 |
| **需要论** | 普尔·克特 | 以用户需要为依据，强调实用性 | 重视用户需求 | 忽视图书馆社会阅读导向作用 |
| **综合计分法** | 麦高文 | 从知识价值和社会需求两方面综合评价 | 具有辩证观点 | 实践中难以实行 |
| **社会调查选择论** | 维拉德 | 依据对读者的社会、心理研究结果选书 | 科学分组分析 | 操作复杂，时效性受限 |

**考试提示**：常考"片面性"的对应关系——价值论忽视需求，需要论忽视导向。

### 2. 金色OA vs 绿色OA

| 比较维度 | 金色OA（Gold OA） | 绿色OA（Green OA） |
|----------|-------------------|-------------------|
| **别称** | 开放出版（Open Access Publishing） | 开放存储（Open Access Archives） |
| **实现方式** | 通过发表在OA期刊上开放获取 | 存储到机构或学科知识库 |
| **两种形式** | 开放期刊（整刊开放）/开放论文（部分开放） | 机构知识库 / 预印本数据库 |
| **费用** | 可能有论文处理费APC | 一般免费自存储 |
| **版权归属** | 版权更多归属于作者 | 依据出版商自归档政策 |
| **开放时滞** | 立即开放 | 可能有 embargo（时滞期） |
| **代表平台** | BMC、PLoS、DOAJ | arXiv、ChinaXiv、中国科技论文在线 |
| **核心区别** | 在OA期刊上发表 | 在知识库中存储已发表/未发表成果 |

### 3. 成本类型比较

| 成本类型 | 定义 | 示例 |
|----------|------|------|
| **会计成本** | 实际发生的成本 | 购书经费、设备费用、人员工资 |
| **机会成本** | 选择某种方案而放弃的其他方案的收益 | 选择购买A学科图书而放弃B学科图书的潜在收益 |

### 4. 文献增长理论 vs 文献老化理论

| 维度 | 文献增长理论（普赖斯曲线） | 文献老化理论 |
|------|---------------------------|--------------|
| **核心问题** | 文献如何增长 | 文献如何失效 |
| **数学模型** | F(t) = ae^(bt) 指数增长 | 半衰期概念 |
| **时间方向** | 向前看（增长趋势） | 向后看（衰退趋势） |
| **对建设的意义** | 提供丰富信息源，但也产生信息污染 | 为评价选择、优化结构、制定原则提供依据 |

### 题目3（简答题）

**简述阅读环境的变化及其对信息资源建设的影响。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、阅读环境的变化**

1. **阅读形态变化**：从纸媒到多媒介、富文本，超文本成为阅读对象（文字+图片+音视频），数字化阅读率提高，从"读书"到"读屏"。
2. **阅读观念变化**：沉浸式阅读与社群社交化阅读相互融合，名著阅读影视化（通俗本/精简本/普及本/图画本/口袋本），听书率上升。
3. **阅读方式变化**：从线性到非线性阅读，从深阅读到浅阅读，呈现碎片化、跳跃式、快餐式特征。

**二、对信息资源建设的影响**

1. 需适应多媒介、富文本的阅读形态，加强数字资源建设
2. 需应对电子书馆配痛点（多终端适配、多元需求满足）
3. 需平衡深阅读与浅阅读的资源配比
4. 需关注听书等新型阅读方式，拓展资源类型

**解析**：本题考查环境扫描中的阅读环境变化。需从形态、观念、方式三个维度阐述变化，并结合信息资源建设实际分析影响。

### 题目4（简答题）

**简述开放获取（OA）的类型及其区别。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**开放获取（Open Access, OA）**是指科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权。

OA主要分为两种类型：

1. **金色OA（Gold OA）**：又称开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取。两种形式：开放期刊（整刊开放）和开放论文（部分开放）。可能有论文处理费APC。代表平台：BMC、PLoS、DOAJ。

2. **绿色OA（Green OA）**：又称开放存储（Open Access Archives），将研究成果存储到机构或学科知识库。两种形式：机构知识库和预印本数据库。一般免费自存储，但可能有 embargo（时滞期）。代表平台：arXiv、ChinaXiv。

**核心区别**：金色OA是"在OA期刊上发表"，绿色OA是"在知识库中存储"。

**解析**：本题考查OA的类型辨识。需先定义OA（强调"免费使用+保留著作权"），再从实现方式、形式、费用、时滞等维度比较金色与绿色OA。

</details>

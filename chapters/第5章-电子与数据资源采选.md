## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **特色数据库** | 图书馆根据本馆特色馆藏、特定用户需求或地域文化特色，自建的具有独特性的数据库 | 网络资源建设的核心内容 |
| **数据资源化** | 数据收集与清洗、数据标注与描述、数据组织与集成，使数据成为可用资源 | 数据价值化第一阶段 |
| **数据资产化** | 数据确权、数据估值、数据管理，使数据成为可计量的资产 | 数据价值化第二阶段 |
| **数据资本化** | 数据保值增值、数据流通交易、数据开发利用，使数据成为可交易的资本 | 数据价值化第三阶段 |
| **FAIR原则** | 科学数据开放共享应实现可发现（Findable）、可访问（Accessible）、可互操作（Interoperable）、可重用（Reusable），4个目标层层递进 | 2014年提出 |
| **政务数据** | 主要存在于政府电子政务系统，是政府公务活动过程中生产的数据所形成的数据资源 | 公共数据 |
| **科学数据** | 自然科学、工程技术科学等领域通过基础研究、应用研究、试验开发等产生的数据，以及通过观测监测、考察调查、检验检测等方式取得的原始数据及其衍生数据 | 公共数据 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **网络信息采选四要求** | "特"（特色资源）/"精"（精品资源）/"全"（完整性）/"省"（利用免费资源） | 特色数据库建设遵循 |
| **企业数据** | 国有控股企业经营活动中形成的数据资源 | 私密数据，如电信/银行/央企 |
| **个人数据** | 与已识别或可识别自然人相关的数据 | 私密数据，直接标识/间接标识 |
| **原始数据** | 直接产生的基础数据 | 观测、监测、调查取得 |
| **衍生数据** | 由原始数据加工而来 | 汇总、分析后的数据 |

## 【重点公式/算法】

### 1. 数据价值化过程（三阶段）★★★

```
数据资源化          数据资产化          数据资本化
（使数据可用）      （使数据可计量）    （使数据可交易）
    │                  │                  │
    ├─ 数据收集与清洗   ├─ 数据确权         ├─ 数据保值增值
    ├─ 数据标注与描述   ├─ 数据估值         ├─ 数据流通交易
    └─ 数据组织与集成   └─ 数据管理         └─ 数据开发利用
```

**记忆口诀**：资源化使数据可用（收清标注组织），资产化使数据可计量（确权估值管理），资本化使数据可交易（保值流通开发）。

### 2. 数据资源类型分类

```
数据资源
├─ 公共数据
│   ├─ 政务数据（政府公务活动产生）
│   │   ├─ 领域数据资源（如医疗健康/农业/科学数据）
│   │   └─ 区域数据资源（如上海/贵州大数据资源）
│   └─ 科学数据（基础研究/应用研究/试验开发产生）
│       └─ 原始数据 + 衍生数据
└─ 私密数据
    ├─ 企业数据（国有企业：电信/银行/央企）
    └─ 个人数据（直接标识/间接标识）
```

### 3. FAIR原则四要素 ★★★

```
FAIR原则（2014年提出）
│
├─ F - Findable（可发现）
│      └─ 数据可被搜索和定位
│
├─ A - Accessible（可访问）
│      └─ 数据可被获取和访问
│
├─ I - Interoperable（可互操作）
│      └─ 数据可与其他系统互操作
│
└─ R - Reusable（可重用）
       └─ 数据可被重复使用
```

**记忆口诀**：F可发现，A可访问，I可互操作，R可重用；四者层层递进。

### 4. 特色数据库建设四要求

```
特色数据库建设四要求
│
├─ "特" ── 特色资源，人无我有
├─ "精" ── 精品资源，人有我精
├─ "全" ── 完整性，系统收录
└─ "省" ── 利用免费资源，经济节约
```

## 【模拟练习】

### 题目1（名词解释）

**请解释"FAIR原则"的内涵。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**FAIR原则**是科学数据开放共享应遵循的基本原则，于2014年提出，要求科学数据实现四个目标，四个目标层层递进：

1. **F - Findable（可发现）**：数据可被搜索和定位，用户能够找到所需数据
2. **A - Accessible（可访问）**：数据可被获取和访问，用户能够访问数据内容
3. **I - Interoperable（可互操作）**：数据可与其他系统互操作，不同系统和平台间能够交换和使用数据
4. **R - Reusable（可重用）**：数据可被重复使用，数据具有足够的描述和许可信息支持再利用

**解析**：本题是高频名词解释与简答题。需准确记忆FAIR四个字母的含义，并理解四者层层递进的关系：先能发现，再能访问，进而能互操作，最终实现可重用。

</details>

### 题目2（简答题）

**简述数据资源建设的三个层级（数据价值化过程）。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

数据资源建设包含数据价值化的三个阶段，层层递进：

1. **数据资源化（使数据可用）**：通过数据收集与清洗、数据标注与描述、数据组织与集成，使原始数据成为可用的资源。这是数据价值化的第一阶段，解决数据的可用性问题。

2. **数据资产化（使数据可计量）**：通过数据确权、数据估值、数据管理，使数据成为可计量的资产。这是数据价值化的第二阶段，解决数据的产权归属和价值衡量问题。

3. **数据资本化（使数据可交易）**：通过数据保值增值、数据流通交易、数据开发利用，使数据成为可交易的资本。这是数据价值化的第三阶段，解决数据的流通和增值问题。

**解析**：本题是高频简答与论述题。需准确记忆三个阶段的名称、核心目标和主要环节。记忆口诀："资源化使数据可用，资产化使数据可计量，资本化使数据可交易。"

</details>

### 题目3（选择题）

**以下属于公共数据的有（ ）**

A. 政务数据　B. 科学数据　C. 企业数据　D. 个人数据

<details>
<summary>点击查看答案与解析</summary>

**答案：AB**

**解析**：数据资源按公共/私密分类：公共数据包括政务数据和科学数据；私密数据包括企业数据和个人数据。政务数据是政府公务活动产生的数据，科学数据是基础研究、应用研究、试验开发产生的数据，两者属于公共数据。企业数据（国有控股企业经营活动中形成）和个人数据（与已识别或可识别自然人相关）属于私密数据。

</details>

### 题目5（简答题）

**简述特色数据库建设的四要求。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

特色数据库建设应遵循四个要求：

1. **"特"——特色资源**：建设人无我有的特色资源，体现数据库的独特性和差异化
2. **"精"——精品资源**：建设人有我精的精品资源，保证数据库的内容质量
3. **"全"——完整性**：系统收录相关领域的文献信息，保证数据库的系统性和完整性
4. **"省"——利用免费资源**：充分利用免费资源，经济节约地建设数据库

**解析**：本题考查特色数据库建设原则。记忆口诀："特精全省"，特是独特性、精是高质量、全是系统性、省是经济性。

</details>

## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **电子资源采选4依据** | 电子资源采选的四个基本依据：馆藏政策、用户需求、资源品质、定价模式 | 比文献采选3依据多出"资源品质""定价模式"两项 |
| **四级数字资源结构** | 永久保存级→服务级→镜像级→链接级，按数字资源重要程度和保存方式分为四个级别 | 唯一性递减，稳定性递减 |
| **永久保存级馆藏** | 保存价值已鉴定、用途得到认可、具有唯一性，且仅定位在某特定图书馆，其他地方不可获得 | 最高级别，唯一性是核心 |
| **服务级馆藏** | 对用户有用且必须用的虚拟馆藏；本馆用户十分需要，但贮存在其他数字图书馆服务器上 | 虚拟馆藏，存于他处 |
| **镜像级馆藏** | 其他数字图书馆馆藏的拷贝，地址可能移动，经常不可存取 | 拷贝性质，不稳定 |
| **链接级馆藏** | 在任何地方均可寄居的信息资料 | 最低级别 |
| **多重价格** | 同一资源以不同组合方式定价，包括全库+学科库、现期+回溯、全库+单品种、数据库+印本4种 | 定价模式第1大类，4种 |
| **捆绑价格** | 将多种产品捆绑销售定价，包括印本捆绑、新产品捆绑、内容重复产品捆绑3种 | 定价模式第2大类，印本捆绑最典型 |
| **歧视性价格** | 根据买方不同情况差别定价，包括按规模/订购量/使用统计/使用需求分级4种 | 定价模式第3大类，4种 |
| **国家采购** | 对与国家经济及社会发展战略目标相关的、教学科研人员和社会公众普遍需要的基础性、综合性数字资源，利用公共财政即政府出资购买的采购模式 | 3型：统购/补贴/平台 |
| **集团采购** | 若干图书馆自愿组成集团，共同推举谈判代表与电子资源提供商谈判价格与使用条款 | DRAA是典型代表 |
| **DRAA** | 高校图书馆数字资源采购联盟（Digital Resource Acquisition Alliance of Chinese Academic Libraries） | 中国高校集团采购核心组织 |
| **单馆自主采购** | 各馆根据本馆需要，自发与信息资源提供商谈判价格并自行签署购买使用协议 | 适用于价格不高而利用率较高的数据库 |
| **数据资源化** | 数据收集与清洗、数据标注与描述、数据组织与集成，使数据成为可用资源 | 数据价值化第一阶段 |
| **数据资产化** | 数据确权、数据估值、数据管理，使数据成为可计量的资产 | 数据价值化第二阶段 |
| **数据资本化** | 数据保值增值、数据流通交易、数据开发利用，使数据成为可交易的资本 | 数据价值化第三阶段 |
| **FAIR原则** | 科学数据开放共享应实现可发现（Findable）、可访问（Accessible）、可互操作（Interoperable）、可重用（Reusable），4个目标层层递进 | 2014年提出 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **资源品质评价三角度** | 从资源内容（全/广/新）、检索系统功能、提供商信誉与服务三个角度评价电子资源品质 | 电子采选4依据之一 |
| **统购型** | 国家利用中央经费统一购买电子资源，全国每台联网计算机均可免费访问 | 中央经费全额，如冰岛 |
| **补贴型** | 政府支付一部分，另一部分由各图书馆支付 | 政府+各馆分担，如加拿大 |
| **平台型** | 国家投资建立统一采购与访问平台，平台费由国家支付，资源购买费由基层馆负担 | 国家出平台费+基层出资源费，如英国 |
| **特色数据库** | 图书馆根据本馆特色馆藏、特定用户需求或地域文化特色，自建的具有独特性的数据库 | 网络资源建设的核心内容 |
| **政务数据** | 主要存在于政府电子政务系统，是政府公务活动过程中生产的数据所形成的数据资源 | 公共数据 |
| **科学数据** | 自然科学、工程技术科学等领域通过基础研究、应用研究、试验开发等产生的数据，以及通过观测监测、考察调查、检验检测等方式取得的原始数据及其衍生数据 | 公共数据 |
| **企业数据** | 国有控股企业经营活动中形成的数据资源 | 私密数据，如电信/银行/央企 |
| **个人数据** | 与已识别或可识别自然人相关的数据 | 私密数据，直接标识/间接标识 |

### 7. 数据价值化过程（三阶段）★★★

```
数据资源化          数据资产化          数据资本化
```

## 【方法比较】

### 1. 电子采选4依据 vs 文献采选3依据 ★★

| 比较维度 | 文献采选3依据（Ch4） | 电子采选4依据（Ch5） |
|----------|-------------------|-------------------|
| **依据数量** | 3个 | 4个 |
| **共有依据** | 馆藏政策、需求（读者需求） | 馆藏政策、需求（用户需求） |
| **特有依据** | 经费条件 | 资源品质、定价模式 |
| **关注重点** | 经费约束、读者覆盖 | 品质评价、定价策略 |
| **差异原因** | 印本文献定价相对简单，主要受经费约束 | 电子资源定价模式复杂（3大类11种），需专门评价品质与价格 |

**记忆逻辑**：文献3依据 = 共有2（馆藏政策+需求） + 经费条件；电子4依据 = 共有2 + 资源品质 + 定价模式。

### 2. 四级数字资源结构对比 ★★

| 级别 | 名称 | 核心特征 | 获取方式 | 稳定性 |
|------|------|----------|----------|--------|
| **第一级** | 永久保存级馆藏 | 保存价值已鉴定、用途得到认可、具有唯一性，仅定位在某特定图书馆 | 本馆永久保存 | 最高 |
| **第二级** | 服务级馆藏 | 对用户有用且必须用的虚拟馆藏，贮存在其他数字图书馆服务器上 | 远程访问 | 较高 |
| **第三级** | 镜像级馆藏 | 其他数字图书馆馆藏的拷贝，地址可能移动，经常不可存取 | 本地镜像拷贝 | 较低 |
| **第四级** | 链接级馆藏 | 在任何地方均可寄居的信息资料 | 链接访问 | 最低 |

### 题目7（简答题）

**简述电子资源三种采购方式及其比较。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

电子资源采购方式主要有三种：

**解析**：本题是高频简答与论述题。需准确记忆三种方式的含义、资金来源、适用类型和典型实例，并理解三者间的互补关系。

### A. 电子资源采选板块

#### 概念掌握检查

- [ ] 能说出电子资源采选4依据（馆藏政策/用户需求/资源品质/定价模式）
- [ ] 能说出电子4依据与文献3依据的对比关系（共有2项+各自特有）
- [ ] 能按顺序说出四级数字资源结构（永久保存→服务→镜像→链接）
- [ ] 能说出各级数字馆藏的核心特征（唯一性/虚拟馆藏/拷贝性质/寄居信息）
- [ ] 能说出定价模式3大类（多重价格/捆绑价格/歧视性价格）及各类包含的方式数（4/3/4）
- [ ] 能列举多重价格4种方式
- [ ] 能列举捆绑价格3种方式，并说出最典型的是印本捆绑
- [ ] 能列举歧视性价格4种方式
- [ ] 能说出资源品质评价3角度（资源内容/检索系统功能/提供商信誉服务）
- [ ] 能说出资源内容评价的"全广新"
- [ ] 能列举电子资源采购3种方式（国家/集团/单馆）
- [ ] 能说出DRAA的全称和含义
- [ ] 能列举国家采购3种类型及典型国家（统购-冰岛/补贴-加拿大/平台-英国）
- [ ] 能说出三种采购方式对应的层次（公共基础层/综合性多学科全文层/专业性边缘性学科层）

#### 方法应用检查

- [ ] 能比较电子采选4依据与文献采选3依据的异同
- [ ] 能比较四级数字资源结构各级的特征和稳定性
- [ ] 能比较三种电子资源采购方式的异同
- [ ] 能区分国家采购三类型的经费来源
- [ ] 能辨别定价模式3大类的归属

### B. 数据资源建设板块

#### 概念掌握检查

- [ ] 能说出特色数据库的概念和建设四要求（特/精/全/省）
- [ ] 能按顺序说出数据价值化三阶段（资源化→资产化→资本化）
- [ ] 能说出各阶段的核心目标（可用/可计量/可交易）
- [ ] 能列举数据资源化的三个环节（收集清洗/标注描述/组织集成）
- [ ] 能列举数据资产化的三个环节（确权/估值/管理）
- [ ] 能列举数据资本化的三个环节（保值增值/流通交易/开发利用）
- [ ] 能说出数据资源的两大分类（公共数据/私密数据）
- [ ] 能列举公共数据的两个子类（政务数据/科学数据）
- [ ] 能列举私密数据的两个子类（企业数据/个人数据）
- [ ] 能说出FAIR原则的提出年份（2014年）
- [ ] 能准确说出FAIR四个字母的含义（可发现/可访问/可互操作/可重用）
- [ ] 能说出FAIR原则四个目标是层层递进的关系
- [ ] 能区分原始数据和衍生数据

#### 方法应用检查

- [ ] 能比较数据价值化三阶段的目标和环节
- [ ] 能辨别不同类型的数据资源归属（公共/私密）
- [ ] 能解释FAIR原则四要素的递进关系
- [ ] 能运用特色数据库建设四要求分析实际案例

### 易错点提醒

| 易错点 | 正确内容 |
|--------|----------|
| 电子采选依据是3个 | 电子采选依据是**4个**（馆藏政策/用户需求/资源品质/定价模式）；文献采选才是3个 |
| 四级数字资源结构最高是服务级 | 最高是**永久保存级** |
| 定价模式是3大类10种 | 定价模式是**3大类11种**（多重4+捆绑3+歧视4） |
| 捆绑价格最典型的是新产品捆绑 | 最典型的是**印本捆绑** |
| 数据价值化是2个阶段 | 数据价值化是**3个阶段**（资源化/资产化/资本化） |
| FAIR原则是3个要素 | FAIR原则是**4个要素**（F/A/I/R） |
| FAIR原则四要素是并列关系 | FAIR原则四要素是**层层递进**关系 |
| 企业数据属于公共数据 | 企业数据属于**私密数据** |
| 科学数据属于私密数据 | 科学数据属于**公共数据** |
| 特色数据库建设要求是3个 | 建设要求是**4个**（特/精/全/省） |
| FAIR原则是2018年提出 | FAIR原则是**2014年**提出 |
| DRAA是国际组织 | DRAA是**中国高校图书馆数字资源采购联盟** |
| 国家采购只有1种类型 | 国家采购有**3种类型**（统购/补贴/平台型） |

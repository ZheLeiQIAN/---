## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **信息资源建设5类著作权问题** | 图书馆在信息资源建设过程中面临的主要著作权问题，包括翻印影印、采购非法出版物、文献复制、馆藏数字化、数据库建设五类 | 5类是常考简答题 |
| **读者决策采购（PDA/DDA）** | 将读者的阅读需求量化成一定指标，作为图书馆文献采购和馆藏建设的决策依据 | 也称DDA（需求驱动采购）、POD（按需购买） |
| **电子资源定价模式** | 数据库商针对电子资源制定的多样化价格策略，可归纳为3大类11种 | 多样性与复杂化 |
| **多重价格模式** | 通过不同角度或标准将数据库分割成不同产品，赋以不同价格，实现市场细分 | 4种细分方式 |
| **捆绑价格模式** | 将两种或两种以上的相关产品捆绑打包定价出售 | 3种捆绑方式 |
| **歧视性价格模式** | 数据库商针对不同图书馆的需求和支付能力制定不同收费价格，形成价格歧视 | 4种分级方式 |
| **资源品质评价三角度** | 从资源内容、检索系统功能、资源提供商信誉与服务三个角度评价电子资源品质 | 采购品质评估框架 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **第三空间（Third Place）** | 由Ray Oldenburg提出，指居住（第一空间）、工作（第二空间）之外的购物休闲空间 | 图书馆定位为第三空间 |
| **期刊危机（Serials Crisis）** | 连续出版物支出增长远超通胀和图书馆预算，导致图书馆无力负担 | 支出占比从25%增至近40% |
| **智慧图书馆** | 基于微服务架构的统一云服务开放平台，以数据共享为纽带连接各功能模块 | "智能"重技术，"智慧"重人本 |
| **征订目录** | 出版发行机构为推销出版物而编制的目录，是图书馆预订采购的主要依据 | 预订采购工具 |
| **通报性书目** | 及时报道最新出版文献的书目，如《全国新书目》 | 报道最新出版 |
| **参考性书目** | 为特定目的而编制的书目，供查阅参考，如国家书目 | 国家书目属此类 |

## 【重点公式/算法】

### 1. 信息资源建设5类著作权问题

```
信息资源建设著作权问题（5类）
│
├─ ① 翻印/影印外国出版物
│      └─ 未经授权翻印影印涉及复制权侵权
│
├─ ② 采购非法出版物
│      └─ 采购盗版或侵权出版物构成侵权
│
├─ ③ 文献复制
│      └─ 超出合理使用范围的复制涉及侵权
│
├─ ④ 馆藏文献数字化
│      └─ 数字化实质为行使复制权，须获授权
│
└─ ⑤ 数据库建设
       ├─ 指南性数据库（馆藏书目/联合书目/特色文献数据库）
       └─ 源数据库（读者可获取原始文献，具独立著作权）
```

**记忆口诀**：翻影非法复制化，数据库建五类查。

### 2. PDA 4种类型（按读者参与途径）

```
读者决策采购（PDA）── 按读者参与途径分类
│
├─ ① 通过馆际互借
│      └─ 对互借请求购买而非借阅（普渡大学，150美元/3年内）
│
├─ ② 通过馆藏/联合目录
│      └─ 书目导入目录，读者检索后荐购（伊利诺伊，300美元/1年内）
│
├─ ③ 通过电子书点击浏览
│      └─ 浏览触发自动购买（丹佛大学，5分钟/10页→短期租借→3次购买）
│
└─ ④ 通过网上/线下书店
       └─ 读者书店发现→图书馆购买（水牛城大学，IP认证）
```

**记忆口诀**：互借目录点击浏览，书店四种途径。

### 3. 采选依据对比

```
文献资源采选3依据          电子资源采选4依据
│                          │
├─ 馆藏政策                ├─ 馆藏政策
├─ 读者需求                ├─ 用户需求
├─ 经费条件                ├─ 资源品质 ★（新增）
│                          └─ 定价模式 ★（新增）
```

**记忆口诀**：文献三依据（政策需求经费），电子四依据（加品质定价）。

### 4. 定价模式3大类11种结构

```
电子资源定价模式
│
├─ 多重价格（4种）
│   ├─ ①全库价格 + 学科库价格（如爱思唯尔SD：全库+24学科库=25种价格）
│   ├─ ②现期数据价格 + 回溯数据价格（如《自然》系列：现刊限最近5年）
│   ├─ ③全库价格 + 单品种价格（如德古意特电子书；OCLC ECO按篇购买）
│   └─ ④数据库价格 + 印本价格（IOP模式 / Wiley模式）
│
├─ 捆绑价格（3种）
│   ├─ ①印本捆绑（最典型常见，如爱思唯尔SD最低订购额）
│   ├─ ②新产品捆绑（打包新产品，涨幅偏高）
│   └─ ③内容重复产品捆绑（如EBSCO ASP与BSP重复率20%）
│
└─ 歧视性价格（4种）
    ├─ ①按学校/机构规模分级（如汤森路透SCI：6个级别）
    ├─ ②按现有订购量分级（如Wiley：内容费越高，访问费越低）
    ├─ ③按使用统计分级（如IEEE按下载量4级；IOP按成本3级）
    └─ ④按使用需求分级（如SciFinder按并发用户数定价）
```

**记忆口诀**：多重四（全库学科、现期回溯、全库单品种、数据库印本），捆绑三（印本、新产品、重复），歧视四（规模、订购量、统计、需求），共计十一。

### 5. 资源品质评价三角度

```
资源品质评价
│
├─ ① 资源内容
│      └─ 全（覆盖范围）、广（学科广度）、新（更新频率）
│
├─ ② 检索系统功能
│      └─ 检索途径、检索字段、检准率与检全率、用户体验
│
└─ ③ 资源提供商的信誉与服务
       └─ 提供商信誉、技术支持、售后服务、培训服务
```

### 6. 采选书目三种类型

```
采选书目类型
│
├─ 征订目录
│      └─ 出版发行机构编制，推销出版物用，图书馆预订采购依据
│
├─ 通报性书目
│      └─ 及时报道最新出版文献，如《全国新书目》
│
└─ 参考性书目

```

**辨识要点**：征订目录→推销用；通报性书目→报道最新；参考性书目→查阅参考。

### 2. 定价模式3大类对比 ★★★

| 比较维度 | 多重价格模式 | 捆绑价格模式 | 歧视性价格模式 |
|----------|--------------|--------------|----------------|
| **核心逻辑** | 将数据库分割成不同产品，赋以不同价格，实现市场细分 | 将两种或两种以上相关产品捆绑打包定价出售 | 针对不同图书馆需求和支付能力制定不同收费价格 |
| **细分数量** | 4种 | 3种 | 4种 |
| **具体类型** | ①全库+学科库 ②现期+回溯 ③全库+单品种 ④数据库+印本 | ①印本捆绑 ②新产品捆绑 ③内容重复产品捆绑 | ①按规模分级 ②按订购量分级 ③按使用统计分级 ④按使用需求分级 |

### 题目1（简答题）

**简述图书馆信息资源建设面临的主要著作权问题（5类）。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

图书馆信息资源建设面临的著作权问题主要有5类：

1. **翻印/影印外国出版物**：未经授权翻印或影印外国出版物，涉及复制权侵权问题
2. **采购非法出版物**：采购盗版或侵权出版物，本身构成侵权行为
3. **文献复制**：超出合理使用范围的文献复制服务涉及著作权侵权
4. **馆藏文献数字化**：馆藏文献数字化实质是行使著作权人的复制权，须获得授权
5. **数据库建设**：包括指南性数据库（馆藏书目、联合书目、特色文献数据库）和源数据库（具独立著作权）建设中的著作权问题

**解析**：本题是高频简答题。需完整列举5类并简要说明。记忆口诀："翻影非法复制化，数据库建五类查。"

### 题目2（简答题）

**简述读者决策采购（PDA）按读者参与途径分类的4种类型。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

读者决策采购（PDA）按读者参与途径可分为4种类型：

1. **通过馆际互借**：当读者提出馆际互借请求时，图书馆直接购买该文献而非借阅。如普渡大学设定150美元/3年内的阈值
2. **通过馆藏/联合目录**：将书商书目导入馆藏或联合目录，读者检索后触发荐购。如伊利诺伊大学设定300美元/1年内的阈值
3. **通过电子书点击浏览**：读者浏览电子书达到设定条件后触发自动购买。如丹佛大学设定5分钟/10页→短期租借→3次触发购买
4. **通过网上/线下书店**：读者在书店发现所需文献后，通过IP认证等方式由图书馆购买。如水牛城大学模式

**解析**：本题考查PDA的4种实施方式。关键在于区分"读者参与的途径"不同。记忆口诀："互借目录点击浏览，书店四种途径。"

</details>

### 题目3（简答题）

**简述电子资源定价模式的3大类11种，并举例说明。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

电子资源定价模式可归纳为3大类共11种：

**一、多重价格模式（4种）**——将数据库分割成不同产品，赋以不同价格，实现市场细分：
1. 全库价格+学科库价格（如爱思唯尔SD：全库+24学科库=25种价格）
2. 现期数据价格+回溯数据价格（如《自然》系列：现刊限最近5年）
3. 全库价格+单品种价格（如德古意特电子书）
4. 数据库价格+印本价格（如IOP模式）

**二、捆绑价格模式（3种）**——将两种或两种以上相关产品捆绑打包定价出售：
1. 印本捆绑（最典型常见，如爱思唯尔SD最低订购额）
2. 新产品捆绑（打包新产品，涨幅偏高）
3. 内容重复产品捆绑（如EBSCO ASP与BSP重复率20%）

**三、歧视性价格模式（4种）**——针对不同图书馆需求和支付能力制定不同收费价格：
1. 按学校/机构规模分级（如汤森路透SCI：6个级别）
2. 按现有订购量分级（如Wiley）
3. 按使用统计分级（如IEEE按下载量4级）
4. 按使用需求分级（如SciFinder按并发用户数定价）

**解析**：本题是高频简答与论述题。需准确记忆3大类名称、每类细分数量及典型实例。记忆口诀："多重四，捆绑三，歧视四，共计十一。"

</details>

### 题目5（名词解释）

**请解释"资源品质评价三角度"的内涵。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**资源品质评价三角度**是电子资源采选中评估资源品质的三个维度框架：

1. **资源内容**：从内容的覆盖范围（全）、学科广度（广）、更新频率（新）三个指标评价资源内容质量
2. **检索系统功能**：评价检索途径、检索字段、检准率与检全率、用户体验等系统功能
3. **资源提供商的信誉与服务**：评价提供商的信誉、技术支持、售后服务、培训服务等

**解析**：本题是电子资源采选依据中的核心知识点。注意三角度的完整性和各自关注焦点的区别：内容看质量、系统看功能、提供商看服务。
